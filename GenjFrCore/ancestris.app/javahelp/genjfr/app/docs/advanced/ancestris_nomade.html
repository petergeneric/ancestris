Ancestris - Nomade


Vous avez envie de vous balader avec toute votre généalogie avec vous, pour la montrer par exemple à un copain qui a un ordinateur, etc... Ancestris a été prévu pour ça également. Comment faire?

Vous êtes sous Windows :
  • On commence par repérer où est sa clé USB dans l'arborescence de vos répertoires.
    Vous devriez voir apparaître votre clé dans votre navigateur de fichiers.
  • Vous créez sur la racine de votre clé, un répertoire que l'on pourrait appeler "ancestris".
    Sous Windows, vous pourriez avoir quelque chose du genre : F:\ancestris (si F: est la lettre attribuée par Windows à votre clé).
  • Dans ce sous répertoire "ancestris", vous placez le fichier que vous venez de télécharger. Il s'appelle en général ancestris-windows.exe. Il vous suffit de le lancer et de demander à ce que tout s'installe sur votre clé USB. Il est possible aussi que vous ayez téléchargé une version intermédiaire, dont le nom est sous la forme ancestris_xxxxx.zip. (xxxx est à remplacer par le numéro de version du fichier que vous venez de télécharger - ex: ancestris_0.2-2.zip ou ancestris-20100226.zip).
  • Vous décompressez le fichier ancestris_xxxxx.zip, il va se créer tous les sous-répertoires nécessaires. Une fois que c'est fait, vous pouvez, pour gagner un peu de place, effacer le fichier ancestris_xxxxx.zip, vous n'en aurez plus besoin. Idem si vous aviez repris le fichier ancestris-windows.exe
  • Vous êtes sous Windows, vous pourriez utilement mettre dans le répertoire "ancestris\bin" de votre clé (c'est là que vous allez trouver un fichier exécutable, qui s'appelle ancestris.exe), un petit fichier bat pour lancer Ancestris. Sous Windows, un fichier bat est un fichier de commande exécutable). Pour vous simplifier la vie, vous ouvrez un éditeur texte, genre Notepad, et vous faites un copier/coller, du texte ci-dessous (les 3 lignes qui suivent et attention qu'elles soient bien sur trois lignes) dans votre éditeur, ensuite vous sauvegardez ce texte en appelant le fichier nouvellement créé "runancestris.bat" (faites "Sauvegarder Sous"). Sur la capture d'écran, il s'appelle simplement "run".
    rem - Ce fichier sert a lancer le logiciel de genealogie Ancestris -
    @echo off
    ancestris.exe --userdir ..\.ancestris
    Voilà un exemple de ce à quoi ressemble l'arborescence des fichiers de la clé USB sous un système Windows :

  • Je vous conseille de créer un sous-répertoire ancestris\gedcom (sous Windows), et dans ce sous-répertoire "gedcom", vous y mettez votre ou vos fichiers gedcom.
  • Vous pouvez également créer un sous-répertoire ancestris\documents (sous Windows), dans lequel seront stockés les documents crées.
  • Vous êtes sous Windows, alors il vous suffit de cliquer sur runancestris.bat, il va créer automatiquement un répertoire .ancestris juste à coté du dossier ancestris.
  • Si déjà vous avez votre configuration aux petits oignons, pourquoi ne pas récupérer tout votre sous répertoire ".ancestris" et installez le sur votre clé. Vous pouvez aussi utiliser la fonction d'importation des paramètres d'Ancestris si vous avez pris la précaution sur votre ordinateur principal, de faire un export vers un fichier de configuration zippé de votre choix. Lors du lancement à partir de la clé USB, si vous avez copié dessus, votre fichier de configuration zippé, vous allez dans les paramètres d'Ancestris, et vous faites l'import en donnant le chemin et le nom du fichier qui se trouve sur votre clé.
    Vous reprendrez ainsi tous les paramétrages, ainsi que tous les fichiers nécessaires à la localisation (visualisation géographique).

Pour lancer maintenant Ancestris vous ne cliquez plus sur l'exécutable mais sur «runancestris.bat» que vous venez d'installer, vous pouvez également créer un raccourci que vous placez à la racine de votre clef USB.

Vous êtes sous Linux :
  • Dès que vous aurez installé votre clé dans l'USB, celle ci sera automatiquement reconnu par votre système et appara√étra sur votre bureau.
    Ouvrez votre clé et créez un nouveau dossier que vous nommerez par exemple «ancestris».
  • Téléchargez Ancestris sur le site d'Ancestris : http://www.ancestris.org.
    Plusieurs possibilités mais on va supposer que vous téléchargé le fichier ancestris-linux.sh. Vous avez pu aussi télécharger un fichier intermédiaire qui est disponible sous la dénomination ancestris_xxxxx.zip. (xxxx est à remplacer par le numéro de version du fichier que vous venez de télécharger - ex: ancestris_0.2-2.zip ou ancestris-20100226.zip). Vous pouvez demander à votre navigateur (Firefox), l'emplacement du téléchargement. Dans «Edition», «Préférences», «Onglet général». repérez la ligne «Enregistrer les fichiers dans le dossier». Cliquez sur parcourir et choisissez le lieu de stockage, dans notre cas, il s'agit de votre clé usb et du dossier ancestris.
    Si vous avez repris le fichier ancestris-linux.sh, il vous suffit d'ouvrir un terminal, de copier ce fichier sur votre clé, et de lancer la commande "sh ancestris-linux.sh".
  • Rendez-vous ensuite dans votre répertoire ancestris de votre clé USB, où vous trouverez le fichier zippé fraichement téléchargé, cliquez droit dessus et descendez sur la ligne «extraire ici». Pour un gain de place vous pouvez sans souci, après extraction, supprimer le fichier zippé.
  • Ouvrez le dossier «ancestris», puis le sous-répertoire «bin» qui contient deux exécutables, un pour windows (ancestris.exe), et un pour Linux (ancestris). Dans notre cas on va prendre le fichier «ancestris». Cliquez dessus, une fenêtre vous demandera si vous voulez lancer ancestris. Il existe deux possibilités, une dans un terminal et le lancement simple, prenez la deuxième possibilité. Vous voilà maintenant dans le logiciel Ancestris.
  • Comme pour Windows vous pouvez créer des sous répertoires «ancestris/gedcom» - «ancestris/documents» sur votre clé USB.
  • Vous pouvez utiliser un lanceur, directement installé sur votre bureau. Clic droit sur un espace vide de celui-ci, créer un lanceur. Dans Nom, mettez Ancestris. Dans commande, ouvrir parcourir, cherchez votre clé /ancestris/ancestris/bin/ancestris et enfin validez. Sur votre bureau un icône nommé Ancestris vient d'être créer, cliquez dessus afin d'ouvrir le logiciel.
