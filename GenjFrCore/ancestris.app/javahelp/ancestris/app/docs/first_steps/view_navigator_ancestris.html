La fenêtre du Navigateur


Le navigateur est la vue qui permet de naviguer le plus efficacement dans votre fichier généalogique via les liens familiaux les plus proches :
  • Dans la partie haute : le nom de l'individu sélectionné est affiché.
  • Dans la partie basse :
    7 boutons sont dessinés (chaque type de catégorie apparaît en passant la souris dessus):
    • l'individu
    • ses frères et soeurs plus âgés
    • ses frères et soeurs plus jeunes
    • son père
    • sa mère
    • ses conjoints
    • ses enfants

      - Si une de ces catégories comporte plusieurs personnes :
         un clic prolongé sur le bouton permet d'afficher la liste des personnes et ainsi sélectionner celle désirée.
         (ex : si l'individu a plusieurs enfants, cela affichera la liste de tous les enfants qu'il a eu avec son conjoint, etc.)

      - Si une de ces catégories ne comporte aucune personne, le bouton est grisé et est non sélectionnable.

    La sélection d'un individu dans cette fenêtre, met à jour les données affichées dans les autres fenêtres:
    • Editeurs
    • Arbre
    • Table
    • Entité
