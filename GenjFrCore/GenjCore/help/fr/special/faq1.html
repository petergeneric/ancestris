I - Questions Générales

  1. Q - Qu'est-ce que c'est que GenJ ?
    R - GenealogyJ (en abrégé GenJ) est un visualiseur et un éditeur de données généalogiques stockées dans le format Gedcom.

  2. Q - Que veut dire le mot Gedcom ?
    R - Lisez les explications ici.

  3. Q - Y a t'il une forum de discussion concernant GenJ ?
    R - Oui, ce forum de discussion existe. Vous pouvez nous y rejoindre ici (forum de discussion@genj.sourceforge.net)
    Il existe également un canal IRC sur le serveur irc.freenode.net canal #genj. D'autres utilisateurs ou/et développeurs peuvent y être présents. Je vous conseille trés vivement d'installer un client IRC sur votre ordinateur et d'y aller.

  4. Q - Comment puis-je être en contact avec les autres utilisateurs de GenJ et les développeurs ? Où puis-je avec avoir de plus amples informations à propos de GenJ ?
    R - Faites un tour sur :
        -  le Site Internet de GenJ,
        -  le Site de développement de GenJ,
        -  et le canal IRC : #genj sur irc.freenode.net.

  5. Q - Comment connaître les modifications apportées chaque nuit dans les versions quotidiennes de développement qu'on appelle les "Nightlies" ? Existe-t-il une liste genre Changelog ?
    R - Les versions quotidiennes se trouvent ici :  Nightlies.
    Attention, ce sont des versions beta, donc, vous les utilisez à vos risques et périls ; n'oubliez pas de faire des sauvegardes de vos données.
    Pour connaître les modifications, il suffit de vous rendre sur ce site. Vous y trouverez le commentaire sur les 20 derniers changements apportés au code.

  6. Q - Qu'est ce qu'il me faut si je veux aider au développement de GenJ ou au moins, commencer à faire des rapports ?
    R - Normalement, il vous faut le dernier JDK disponible sur le site de Sun.
    Ensuite, il serait bien que vous récupériez la dernière archive contenant le Javadoc (le Javadoc n'est pas indispensable si vous utilisez un outil comme Eclipse (voir plus loin), mais il me semble vraiment utile de savoir à quoi ressemble l'API de GenJ). Pour cela allez voir sur le site de GenJ dans la zone de téléchargement, ou si il n'y est pas, vous pouvez le trouver dans les Nightlies.
    Enfin, utilisez un outil de développement, comme Eclipse. Pour le reprendre, allez sur ce site, et prenez la version qui correspond à votre environnement. Eclipse est ce qu'on appelle un "Java IDE" (Java Integrated Development Environment), donc un éditeur de développement.
