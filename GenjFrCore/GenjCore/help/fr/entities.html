Les Entités

GenJ vous permet de présenter les entités suivantes (en suivant le standard Gedcom):

 Individu
 Famille
 Média
 Note
 Source
 Fournisseur d'Information
 Lieux de Stockage

Ces entités sont ce que le format Gedcom appelle des enregistrements.
Seuls ces 7 types sont autorisés. Il vous convient donc de faire entrer vos données dans ce cadre.

Pour créer des entités, faites un clic droit sur n'importe quelle vue pour ouvrir un Menu Contextuel. A partir de là, vous pouvez choisir d'ajouter une entité pour
  1. une propriété ou
  2. une entité ou
  3. un fichier Gedcom
Cependant sachez que la présence des sous menus 1-3 dépend de l'endroit sur lequel vous avez cliqué. Si vous choisissez de créer ou d'ajouter une entité ainsi qu'il a été mis ci-dessus, un écran va apparaître et vous permettra de confirmer si l'action entreprise doit être exécutée, ou au contraire abandonnée (Il vous sera également possible de lier votre action avec d'autres éléments de votre arbre généalogique).
