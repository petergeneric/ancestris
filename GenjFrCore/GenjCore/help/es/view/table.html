Vistas - Vista de Tabla

Esta Vista muestra las entidades en filas - las columnas son propiedades de esas entidades (p.e. Nombre , Sexo y fecha de nacimiento) Se puede hojear la tabla con las teclas del cursor o la barra de desplazamiento. Para ordenar las filas por una columna determinada simplemente haga clic sobre la cabecera de esa columna. Un pequeño marcador dirá la dirección de ordenamiento.

¿Ahora bien como se calculan los valores de cada celda? La tabla sólo muestra un tipo de entidad cada vez. Cada fila muestra una instancia del tipo mostrado. Las celdas en cada fila son valores de una propiedad de una entidad identificada por Etiquetas. La ruta de etiquetas para una columna se muestra en la cabecera (p.e. INDI:BIRT:DATE es la fecha de nacimiento del individuo contenido en una fila).

Configuración

Las configuraciónes para esta vista son agrupadas por tipo de entidad. Para cada tipo ( seleccionado arriba ) las columnas mostradas son marcadas en el centro (deseleccione nodos en el centro ). El orden de la columna de izquierda a derecha se muestra en la parte de abajo (use los botones a la izquierda para reordenarlos ).
