Centro de Control

La primera ventana que se ve al iniciar GenJ muestra una lista de ficheros Gedcom abiertos. La primera vez que se usa se debe abrir automaticamente un fichero llamado 'example.ged' . Usted puede explorar las características de GenJ abriendo una vista de ese fichero o puede crear el suyo. Los botones en la parte de arriba le permitirán:

 Abrir un fichero existente o crear uno nuevo
 Abrir una Vista de Tabla*
 Abrir una Vista de Árbol*
 Abrir una Vista de Línea temporal*
 Abrir una Vista de Edición*
 Abrir una Vista de Informe*
 Abrir una Vista de Navegador*
 Abrir una Vista de Entidad*
 Abrir una Vista de Búsqueda*

*( Asegurese de que el fichero seleccionado en la lista es el fichero sobre el que va a trabajar )

Usted puede abrir tantos ficheros como quiera - serán numerados secuencialmente y cada uno guardará su propia configuración y preferencias en caso de que quiera verlos desde "diferentes ángulos"
