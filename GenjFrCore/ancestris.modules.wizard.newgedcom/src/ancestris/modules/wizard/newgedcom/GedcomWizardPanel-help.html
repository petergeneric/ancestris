La gestion des Juridictions

Ici on aborde une question qui est très importante et qui va avoir des conséquences sur votre généalogie, tout au long de votre travail.
Il importe d'y attacher la plus grande attention, car une erreur dans votre choix vous génera considérablement plus tard. Si vous voulez par la suite corriger vos erreurs sur cette question, cela vous demandera un gros travail.
Vous le comprenez certainement: Attention à cette gestion des juridictions.

Qu'est ce qu'une juridiction?
Une juridiction est un élément qui défini un endroit au sens juridique du terme.
Ainsi une juridiction sera une commune, un département une région, un pays, mais aussi un code postal un code insee, une adresse postale (rue), un état au sens américain du terme, etc.

Dans quel ordre doit-on les mettre? La norme en matière de généalogie, est la norme gedcom. Celle-ci nous impose de mettre les juridictions dans un ordre croissant. Ainsi vous ne pouvez pas présenter vos juridictions de la manière suivante:
lieudit - pays - ville - département
Mais au contraire, vous devez les présenter comme suit:
lieudit - ville - département - pays

Choix des juridictions

Tout dépend du pays qui sera le plus présent dans votre généalogie. Si c'est la France, voilà quelques exemples de juridictions qui sont importantes et dont le choix est judicieux.
  • lieudit
  • commune
  • code_postal
  • code_insee
  • département
  • région
  • pays

Ordre

Respectez l'ordre imposé par la norme : du plus petit au plus grand. Votre première juridiction dans la liste doit être la plus petite. Dans mon exemple, c'est le lieudit. La plus importante sera le pays.

Orthographe

On vous donne le conseil de ne pas mettre de blanc dans un nom de juridiction. Cela n'est pas imposé par le norme gedcom, donc libre à vous de faire différemment. Ainsi "code postal", sera écrit "code_postal".

Conseils

N'hésitez pas à choisir plus de juridictions au départ. En effet, au début on peut trouver que mettre le code insee est superflu. Choisissez le quand même car même si vous ne renseignez pas le code insee pour vos différentes localités, votre fichier gedcom (fichier de généalogie) aura réservé l'emplacement si un jour vous complétez votre travail.
Ne choisissez pas des juridictions trop restreintes dans le temps, exemple une communauté de communes. Vous pouvez vous dire que le code postal ou le code insee, sont des créations récentes et que ces deux juridictions ne vous serviront à rien. Détrompez-vous car elles vous seront utiles par la suite notamment quand il s'agira de localiser les personnes de votre généalogie sur une carte. Les fonctionnalités d'Ancestris dans ce domaine sont nombreuses, et notamment certaines d'entre elles interrogent un serveur internet pour avoir les coordonnées géographiques d'une localité pour pouvoir les afficher sur une carte. Les bases de données de ces serveurs peuvent se servir des codes postaux, des codes insee.
