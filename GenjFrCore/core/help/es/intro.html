GenJ es un visualizador y editor de datos genalógicos que está escrito en Java 100% . Usted está invitado a usarlo libremente, a desgargar el código fuente o a unirse al equipo y contribuir a este programa.

Esta ayuda en línea le proporciona un punto de partida sobre las características de Genj y le enseñará sobre sus interioridades y aspectos ocultos :). Usar una herramienta tan poderosa para trabajar con datos genealógicos como GenJ es estupendo pero algunas cosas no son tan fáciles de encontrar para los usuarios noveles. Por favor lea la sección sobre Conceptos generales primero y tendrá la base necesaria para trabajar con Vistas que es con lo que tendrá que interactuar.

GenJ es software libre; usted puede redistribuirlo y/o modificarlo bajo los términos de la Licencia Pública General GNU tal como fue publicada por la Free Software Foundation.

Para las últimas novedades, y/o si está interesado en este proyecto y quiere ayuda por favor visite http://genj.sourceforge.net.

Diviértase!
