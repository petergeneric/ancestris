Arbeiten mit Ansichten

Wenn Sie eine Gedcom-Datei (*.ged) im Kontrollzentrum markiert haben, können Sie Ansichten öffnen.

Die genealogischen Daten können in verschiedenen Ansichten dargestellt werden, zum Beispiel:

  • als Baum und Tabelle, um durch bestehende Informationen zu blättern
  • als Tabelle und im Editor, um neue Informationen einzugeben oder bestehende zu bearbeiten
  • in der Report-Ansicht, um Reports die verschiedensten Auswertungen

Alle Ansichten einer Datei arbeiten auf den gleichen Daten – dies bedeutet:

  • Änderungen, die in einer Ansicht durchgeführt werden, werden sofort in anderen Ansichten wirksam
  • Selektionen in einer Ansicht werden auch in anderen Ansichten wirksam
  • das Schließen der Datei, schließt alle Ansichten dieser Datei
  • Kontextmenüs für angezeigte Informationen bieten Aktionen in allen Ansichten an.

Mit jedem Klick auf eine der Schaltflächen öffnen Sie ein Fenster der entsprechenden Ansicht. Normalerweise werden Sie nur ein Fenster pro Ansichtstyp öffnen, aber Sie können auch durch einen weiteren Klick z.B. ein zweites Editorfenster öffnen und die Daten zweier Personen gleichzeitig betrachten. Dazu mit dem Pinn-Nadel-Knopf ein Fenster auf die eine Person festpinnen und im anderen Fenster zu einer anderen navigieren.

Sie sollten sich die Fenster so einrichten, dass Sie gut mit ihnen arbeiten können. GenJ speichert sich für jede Datei Position und Größe der Fenster und und stellt sie beim nächsten Öffnen wieder her.
