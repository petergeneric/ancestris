Les Entités - Objet Multimédia

Le genre Objet Multimédia, peut contenir différents types de données comme des images, des documents scannés, des enregistrements. Un objet de type média peut être créé une fois et avoir plusieurs autres entités qui lui font référence du fait des données qu'il contient.

Note Importante: Ce type d'entité est intégré dans GenJ, seulement pour être compatible avec la norme Gedcom. Il est fortement déconseillé de créer de tels types d'objets car les informations binaires qu'il contient, sont intégrées dans le fichier Gedcom (.ged). Cela peut entrainer un fichier d'une taille exorbitante (car contenant toutes les données encodées en un format uuencode). Vous devriez plutôt utiliser la Propriété Multimédia que vous pouvez ajouter par l'intermédiaire du menu de la Fenêtre d'Edition. Une Propriété Multimédia est définie par une sous-propriété (FILE) qui peut contenir le nom du fichier, plutôt que les données binaires du fichier lui-même.

Etiquette Gedcom : OBJE
Information Gedcom : S'applique à un ensemble d'attributs, généralement utilisé pour décrire quelque chose. Cela fait souvent référence aux données qui servent à représenter un objet multimédia, comme un enregistrement audio, la photographie d'une personne ou l'image d'un document.
