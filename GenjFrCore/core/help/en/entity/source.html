Entities - Source

Like Submitter and Repository this type keeps track of where genealogical information came from. A Source represents the origin of information like a book or registry.

Alternatively one can use the Source Property inside other entities' property hierarchy by adding instances to the property of choice (see Edit View's toolbar).
