Views - Timeline View

This view shows events stored in a Gedcom file from left to right on a timeline. It starts at the earliest point in time of an event reaching to the latest event. The Gedcom specification defines specific properties for Individuals and Families that are considered to be events (identified through Tags like BIRT, DEAT or GRAD).

The header contains year indicators distributed evenly from left to right. The events consist of a little marker, a symbol, text (optional) and the date when the event happened (optional).

Events are broken into several lines when they might overlap otherwise. Use the scrollbars at the right and the bottom to browse the information.

Toolbar

  1. change the horizontal size reserved for each event
  2. open settings for view
  3. close view

Settings

The settings for this view are grouped into 2 sections

  1. Main - events to be displayed and layout options
  2. Colors - colors of text rendered
