L'esploratore Gedcom


L'esploratore Gedcom vi permette di navigare tra i diverso files Gedcom ; riassume le entità e vi informa sul loro contenuto.
Una particolarità di questa finestra è che essa non può essere disancorata, Non può essere chiusa. La sua posizione naturale è di essere nascosta, ma potete installarla come bottone. (cfr. La Gestione delle finestre).




  • Un menu vi propone 2 tipi di informazione :
    • Tutte le entità;
    • Parentela prossima (questa funzione sarà attivata in una prossima versione di Ancestris).

L'arborescenza
Per ciascuno dei vostri files Gedcom aperti, ogni riga (o livello di entità) può essere sbloccato per fornire il dettaglio del suo contenuto :

  • Il numero di Individui
    • la lista, in ordine alfabetico, di tutti gli individui,
  • Il numero di Famiglie
    • la lista, in ordine alfabetico, di tutte le famiglie,
  • Il numero di Fonti
    • il dettaglio di tutte le fonti create,
  • Il numero di luoghi di conservazione
    • il dettaglio di tutti i luoghi di depoisito registrati,
  • Il numero di Note **
    • il dettaglio delle vostre note,
  • Il numero di fornitori delle informazioni
    • il dettaglio dei vostri fornitori delle informazioni,
  • Il numero di elementi multimediali
    • il dettaglio degli elementi multimediali usati.


**  Le Note qui repertoriate sono solamente le "note entità" (generate con un clic destro su una entità poi "Gedcom xxxxxx.ged", poi "Creare una Nota"), eccetto le "note sotto elemento di una entità" o "note sotto-elemento di una proprietà di una entità" (nota collegata a BIRT, MAR o altro).
Queste note entità possono essere utili nel caso in cui una informazione sia ricorrente o che esse si riferiscano a più individui (es: filiazione da verificare).
