Téléchargement d'Ancestris


Le téléchargement d'Ancestris recoupe deux choses :
  • Le téléchargement proprement dit,
  • Le téléchargement des mises à jour et des modules additionnels.

I - Le téléchargement proprement dit


Si vous lisez ces lignes, c'est qu'à priori vous avez réussi à télécharger Ancestris.

Pour résumé, vous avez plusieurs manière de télécharger une version d'Ancestris qui vous convienne.

Sur le site internet d'Ancestris, à l'adresse suivante :
http://www.arvernes.com/wiki/index.php/Ancestris, dans la zone de téléchargement, vous trouverez des fichiers sous plusieurs formats, en fonction de votre système d'exploitation (Windows, Linux, Mac OS).

  1. Vous êtes sous Windows : Un fichier au format .zip contient tout ce qu'il vous faut pour installer Ancestris.

  2. Vous êtes sous Ubuntu - Linux : Un paquet au format debian (.deb) est disponible. Il vous suffit de le télécharger et de cliquer dessus pour qu'il s'installe tout seul.

  3. Vous êtes sous Fedora, Mandrake, Redhat - Linux : Un paquet au format RPM (.rpm) est téléchargeable. Ce fichier peut être en retard de quelques versions intermédiaires, par rapport à la toute dernière disponible, mais les mises à jour sont bien suivies en général.

  4. Vous êtes sous Linux (d'une manière très générale, donc même si vous avez un système Ubuntu, Fedora, etc....), l'archive au format .zip dont il est question pour Windows, vous permet d'installer Ancestris sur votre ordinateur.

  5. Vous êtes sous Mac OS, l'archive .zip est disponible pour tous ceux sous Mac.

II - Les mises à jour


Trois méthodes :
  • Mise à jour manuelle,
  • Mise à jour automatique,
  • Ubuntu : mise à jour par le dépôt Arvernes,

    • Mise à jour manuelle :
    Les mises à jour sont possibles sur le site officiel de Ancestris.

    Mise à jour automatique :
    Une autre méthode, bien plus simple, vous est offerte par le logiciel lui-même, et ce, quelque soit votre système d'exploitation (Windows, Linux, Mac) si vous avez installé le fichier au format .zip.
    Si vous allez dans "Outils" - "Plug-ins" - "Paramètres", vous pouvez demander au logiciel de vérifier pour vous, si des mises à jour sont disponibles. Cette vérification peut être faite suivant l'intervalle que vous allez définir (tous les jours, toutes les semaines, à chaque démarrage, etc...).
    Si cette option est activée, vous verrez à un moment le logiciel vous signaler qu'il y a x mises à jour de disponibles. Il vous proposera de les télécharger, et de les installer. Lors de cette installation vous devrez accepter la licence et confirmer si le fichier n'est pas signé, son décompactage sur votre ordinateur. Le rédémarrage du logiciel se fera tout seul à la fin de l'installation.

    Si vous avez utilisé un fichier spécifique à votre système d'exploitation, comme Ubuntu, il faut mieux laisser votre ordinateur gérer les mises à jour, et donc désactiver la mise à jour automatique. Rien ne vous empêchera de vérifier de temps en temps si un nouveau plugin est disponible, mais cette question des plugins est traitée ci-après.

    Ubuntu :
    Si vous êtes sous Ubuntu, et que vous avez téléchargé le paquet spécifique à cette distribution, à savoir le fichier debian (.deb).
    Pour que la mise à jour soit gérée automatiquement par votre ordinateur, il vous faut mettre le site Arvernes dans vos dépôts. Voici comment faire :

    Etape 1 : Importation de la clé GPG d'Arvernes.

    Il faut commencer par télécharger la clé gpg qui valide le paquet que vous allez reprendre. Cela vous assure que le logiciel est bien de moi et qu'il n'a pas été trafiqué par quelqu'un. La méthode la plus simple est de taper dans un terminal :
    wget -q http://ubuntu.arvernes.dyndns.org/29E67D70.gpg -O- | sudo apt-key add -
    (Faites un copier-coller de la ligne ci-dessus dans un terminal. Ce sera plus simple. Pour ouvrir un Terminal, vous faites "Applications" puis "Accessoires", puis dans le menu déroulant, vous trouverez "Terminal". Sinon sur le forum d'Ubuntu, on vous explique comment faire).

    Etape 2 : Ajout d'Arvernes dans les dépôts.

    Ensuite, on va ajouter Arvernes dans le site des dépôts offrant des packages pour Ubuntu, pour cela il vous faut suivre la méthode suivante :

    1) Variante Gnome (Ubuntu) :

    • Cliquez sur "Système",
    • Ensuite "Administration",
    • Puis "Source de logiciel",
    • Vous entrez votre mot de passe, car vous allez faire une opération nécessitant les droits de super utilisateur.
    • Vous allez dans l'onglet "Autres logiciels", et là vous faites "Ajouter".
    • Dans le champ "Ligne APT" qui s'ouvre vous mettez :
      deb http://ubuntu.arvernes.dyndns.org hardy main

      N.B. : Même si vous utilisez une version supérieure à Hardy (8.10), ne modifiez pas la ligne ci-dessus la ligne ci-dessus, ça ne risque rien. C'est simplement pour dire à Synaptic qu'il faut qu'il reprenne l'archive dans un répertoire Hardy sur le dépôt. Ancestris est un programme tournant sous Java, donc il ne changera rien sur votre système. Il va s'installer dans son répertoire et c'est tout. Il ne dépend pas de votre version. Si vous mettez Intrepid vous ne pourrez pas reprendre le paquet. Dans l'avenir si j'ai du retour sur ce paquet Ubuntu, je verrais pour faire quelque chose spécifique à Intrepid.

    Vous n'avez plus qu'à sauvegarder en cliquant sur Ajouter une source de mise à jour. Tapez après sur Fermer. Un écran va apparaître qui va vous dire Les informations sur les logiciels disponibles sont obsolètes, ... ; là, vous cliquez sur Actualiser. Si vous ne le faites pas, les packages disponibles sur Arvernes ne figureront pas dans la liste des paquets présents dans Synaptic.

    2) Variante KDE (Kubuntu) :

    • Cliquez sur "Applications",
    • Ensuite "Système",
    • puis "Gestionnaire de paquets Synaptic",
    • puis sur "Configuration", "Dépôts".
    • Allez dans l'onglet "Sources de logiciels", faites "Ajouter",
    • Entrez l'adresse :
      deb http://ubuntu.arvernes.dyndns.org hardy main
    • Fermez, rechargez et voilà.
    Il y a peut-être plus simple mais j'ai trouvé tout de suite comme cela (N'oubliez pas de mettre à jour la clé publique auparavant.

    3) Pour toutes les variantes (Ubuntu, Kubuntu) :

    Si une erreur apparait, du genre :
    W: GPG error: http://ubuntu.arvernes.dyndns.org hardy Release:
    Les signatures suivantes n'ont pas pu être vérifiées car la clé publique n'est pas disponible : NO_PUBKEY EBC74A5629E67D70
    c'est que vous n'avez pas importé la clé publique d'Arvernes. Je vous conseille de refaire l'étape ci-dessus, "Importation de la clé GPG d'Arvernes".

    Etape 3 : Importation d'Ancestris proprement dit.

    Maintenant vous ouvrez Synaptic, donc :
    • Cliquez sur "Système",
    • Puis on descend sur "Administration",
    • Dans le sous-menu qui s'ouvre, on choisit : "Gestionnaire de paquets Synaptic",
    • On vous demande votre mot de passe car vous allez faire une opération nécessitant les droits de super utilisateur.
    • Dans la fenêtre de Synaptic, vous cliquez sur "Rechercher",
    • Pour faire simple, on va simplement taper comme mot recherché "ancestris",
    • Là, miracle, crac, notre petit Ancestris apparaît. Il suffit de le cocher et de demander à ce qu'il s'installe.

    Par la suite, tout sera automatisé. Vous serez automatiquement averti dès qu'une nouvelle version sera disponible sur le site Ubuntu d'Arvernes.
