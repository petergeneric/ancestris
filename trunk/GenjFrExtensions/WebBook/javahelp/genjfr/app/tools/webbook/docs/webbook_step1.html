Assistant - Panneau "Page d'accueil"


Le livre Web génére une page d'accueil à votre site de généalogie. Cette page comporte un menu et un résumé des principaux éléments ou statistiques de votre arbre.

Les options de ce panneau de l'assistant vous permettent de personnaliser le titre et les renseignements concernant l'auteur, d'indiquer un message particulier, ou d'afficher certaines informations statistiques.
