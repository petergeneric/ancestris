Configuration - Send Translation

Select from the main menu: Tools -> Options.

Once the option dialog is displayed, select the "Send Translation" Tab.


You have to indicate:

  • Your Name and First Name - (ex: John Doe)
  • Your email address - (ex: JohnDoe@mydomain.com. Using gmail, it could be JohnDoe@gmail.com).
  • The SMTP outgoing server address. These are examples of some ISP (Internet Service Provider):
    - Free: "smtp.free.fr"
    - La Poste: "smtp.laposte.net"
    - Gmail: "smtp.gmail.com". Using Imap, you might have to write "imap.googlemail.com".
  • The connection encryption type. Examples:
    - Free: None
    - La Poste: None
    - Gmail: TLS
  • SMTP Port - (ex: 25 or 465 or 587 ...)
    The SMTP port will change according to the encryption type (usually 25 for None, 587 for TLS and 465 for SSL), however if your provider requests a specific port number, fill the SMTP port field with the one requested. It has been reported to make Gmail to work, to take those settings: smtp address: smtp.gmail.com, Encryption: TLS, and smtp port: 587. For LaPoste.net the mail host port is 25.
  • Does the server need an authentification? if so (for Gmail for example, it requires an authentification), fullfill the two fields with the appropriate information.

Note: Please send us the parameters that work for you. That way we will add those examples here, to help other users. Thank you.

It doesn't work?
If you cannot send your translations from the Trancestris program, check out this:
Many software firewalls - a firewall program that reside in your computer and filters your Internet traffic - are configured to block all transmissions on port 25 that do not come from the normal mail program. That is to stop mail worms and similar malware programs from transmitting spam.
You will then have to instruct the firewall, to allow port 25 transmission from your Trancestris program to pass through to the outside world. How this is done, is dependent on the make of the firewall, there is no consistent way to do this. You have to consult your manual to find the right settings, and where to do them.
