Configuration

Configuration of Trancestris is very easy.

You can access it from the Main Menu: Tools -> Options. Three tabs are offered: General - Send Translation - Language.

The first time you started Trancestris, most of the things you had to configure have been done, as you have been asked to configure how your computer can send your translation if you did one.
If you need to change something to those settings, check the "Send Translation" (Mail) parameters.

Another setting is the language you want Trancestris to use when you run it. Be careful, it's not the language you want to translate to, it's just the language Trancestris has to speak when you use it.

The last parameters are gathered on the first tab of the settings Menu. Do you use a proxy for your connection to the Internet? Set them up accordingly.
