The Search Function

Trancestris gives you the opportunity to search for a word in the source language, but also in the destination language (the language you want to translate to).

The search function is a two parts search.
When you search for a word, it searches first in which resources files the word is present. Once the list of all the bundles files which match the search, is given and displayed, you will have the possibility to search into each properties resources file, you want.
Two things to remember: When you're looking for a word, you have to take care to the way you spell it, and second: the search can be "case sensitive" too. Looking for "genealogy", is not the same than looking for "Genealogy", if you have activated the case sensitive option.

The best way to understand how it works, is to give an example.
Let say you are translating from english to swedish.

First search: you're looking for the word "Thanks".
So let's go. Click on the icon which represents a magnifying glass.



A window pops up. Do you want to do a "case sensitive" search? If so, don't forget to check the box "Case sensitive", on the left of the window which pops up.
Write the word "Thanks", in the appropriate field. Once done, click on "Search".
Result: a message says : "No occurrence of "Thanks" found." Strange, isn't it? No, there is no word "Thanks" in the english resources files, that's all.
Now, uncheck the option "Case sensitive". The word "Thanks" is always where you typed it. Click on "Search". Bingo, there is a resource file where the word "thanks" is present. Which one? the window gives you the information, we find the word "thanks", in the "AncestrisCore/ancestris.app/src/ancestris/app" resources file.



What to do next? Just double click on the line where the word seems to be present, so on "AncestrisCore/ancestris.app/src/ancestris/app".
Once done, you will see on the main window of Trancestris, that it highlights the right resources file.



On that main window, you just have to click on the button "Next" on the right of the menubar. Trancestris will focus on the sentence where the word you're looking for, is present. It's magic ;-)




Second search: We want to search for the word "Geografisk" but in swedish.
Click on the magnifying glass on the top main menu. A window pops up. Type the word "Geografisk" in the blank field, check the option "Case sensitive" search, on the left, and click on the "Swedish button". Once done, click on "Search". The screen will look like this one :



On this screen you can see that the word you're looking for, the word "Geografisk" is present in two resources files.
Double click on the first line, which is the first occurrence found, the one which is named "AncestrisCore/core/src/genj/gedcom". Then do as it is explained above.
You want to see the second file for the word you're looking for? Easy, double click on the line "AncestrisExtensions/Geo/src/ancestris/modules/geo".
