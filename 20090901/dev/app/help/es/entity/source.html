Entidades - Fuente

Como Remisor yRepositorio este tipo de entidad dice de dónde viene la información. Representa el origen, como un libro o un registro.

Alternativamente se puede usar la propiedad fuente dentro de otras entidades añadiendo instancias a la propiedad de elección ( ver la barra de herramientas de Edit View ).
