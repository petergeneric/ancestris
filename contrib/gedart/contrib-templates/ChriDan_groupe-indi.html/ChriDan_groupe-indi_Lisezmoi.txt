ARBRE DES RELATIONS FAMILIALES

Le rapport "arbre des relations familiales" d'un individu"  affiche une vue graphique des parents et grands-parents, frères, soeurs, mariage(s) et enfants d'un individu.

INSTALLATION : Pour rendre le rapport actif, décompressez-le dans le répertoire "contrib-templates" de "/.genj/gedart".
ATTENTION, ce rapport nécessite pour se lancer de se positionner sur l'individu sélectionné, clic-droit, "Indivu --> Fiches de renseignements". puis dans la liste déroulante, choisir "ChriDan arbre des relations familiales de l'individu sélectionné", sélectionner le répertoire de destination, taper un nom suivi de l'extension "html", cliquer OK.  (il faut auparavant avoir pensé à sélectionner dans "Rapports --> Options --> Montrer le résultat dans un fichier"), 

Ouvrez alors le fichier obtenu dans Firefox.Vous pouvez l'imprimer (dans les options de l'impression, désactiver l'impression des en-têtes, page, etc...". vous pouvez également varier dans les "préférences" de Firefox, avec la taille de la police, pour avoir une mise en page correspondante à vos souhaits. Pour l'envoyer à  vos correspondants, il est conseillé de le transformer en fichier *.pdf, lisible sans déformation par tous. Pour cela, aller dans "imprimer", choisir "imprimer dans un fichier", indiquer un nom et l'extension *.pdf, désactiver aussir les option d'impression des en-têtes et divers.

(texte du 3/11/2009 : Daniel Durand)
