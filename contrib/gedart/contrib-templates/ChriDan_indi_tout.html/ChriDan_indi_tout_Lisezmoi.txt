FICHE "INDIVIDUELLE" 

La fiche "individuelle totale" affiche des renseignements completssur un individu de votre base. Elle affiche s'ils sont présents :
- la photo de l'individu (si OBJE est sous INDI)
- la ou les photos des résidences (si OBJE est sous RESI)
- pour indiquer le chemin des photos : cliquez dans l'éditeur Gedcom sur les "..." à droite de la ligne "FILE", cela ouvre une fenêtre de navigation, vous permettant d'aller jusqu'à votre fichier et de l'indiquer. Vérifiez ensuite que la photo s'affiche dans GenJ.
- le texte des citations de sources (transcriptions d'actes) si elles sont indiquées sous la forme suivante, que je préconise : EVENT (ou BAPM, BIRT,BURI, DEAT) ---> SOUR ---> DATA --> TXT
(voir le tutoriel "Débuter avec GenJ" pour revoir ma méthode d'installation des sources).

INSTALLATION : Pour rendre le rapport actif, décompressez-le dans le répertoire "contrib-templates" de "/.genj/gedart", ouvrez alors votre GenJ, cliquer droit sur un individu, choisir dans la liste des rapports "Fiches de renseignements", puis dans la liste déroulante, choisir "renseignements (ChriDan) sur l'individu", GenJ vous demande un nom de fichier (il faut auparavant avoir pensé à sélectionner dans "Rapports --> Options --> Montrer le résultat dans un fichier"), sélectionner le répertoire de destination, taper un nom suivi de l'extension "html", cliquer OK. 
Ouvrez alors le fichier obtenu dans Firefox (pour avoir à chaque fois, une ascendance par page, choisir dans les "Préférences" de Firefox, la police réglée sur "sérif 9".).Vous pouvez l'imprimer (dans les options de l'impression, désactiver l'impression des en-têtes, page, etc...". Pour l'envoyer à  vos correspondants, il est conseillé de le transformer en fichier *.pdf, lisible sans déformation par tous. Pour cela, aller dans "imprimer", choisir "imprimer dans un fichier", indiquer un nom et l'extension *.pdf, désactiver aussir les option d'impression des en-têtes et divers.

(texte du 30/09/2009 : Daniel Durand)
