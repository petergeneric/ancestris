Assistant - Panneau "Individus"


[Panneau précédent][Panneau suivant]

Ce panneau permet de personnaliser plusieurs éléments liés aux individus :
  • Le choix de l'individu de-cujus (individu principal). Cet individu sera accessible à partir de la page d'accueil, et c'est à partir de lui que seront indiqués les numéros sosa et le rapport d'ancêtres.
  • La façon d'afficher les individus dont le nom est inconnu, dans toutes les sections du livre Web.
  • Les informations à afficher ou pas pour chaque individu de la section détaillée, comme sa famille, ses enfants, ses parrains et marraines, les notes rattachées, etc.
  • L'affichage ou pas du numéro d'identification Ancestris après le nom de chaque individu, dans toutes les sections du Livre.
  • Les critères de définition des données privées. Le bouton permet de modifier les préférences de données privées d'Ancestris comme si vous y accédiez depuis le menu "Préférences". Les données privées seront masquées dans toutes les sections du Livre Web.
    Si vous générez votre livre Web en langage PHP (voir panneau), la saisie des codes que vous aurez choisis permettra de les voir.
