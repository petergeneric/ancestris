Module Géo
Présentation Générale


Le module Geo d'Ancestris permet de placer les individus et les événements de votre généalogie sur une carte géographique.

Le module Géo comporte deux interfaces distinctes qui partagent la même liste de lieux, celle de votre généalogie.
  • La vue géographique : vous voyez dans cette vue une carte géographique où se trouvent tous les lieux de votre généalogie. Vous pouvez paramètrer ceux que vous voulez voir et comment les voir.
  • La liste des lieux : vous pouvez éditer tous les lieux de votre généalogie, et notamment vérifier que les lieux ont été correctement saisis en terme de juridictions.

Utilisation de la Vue Géographique

L'utilisation de la Vue Géographique est très simple. Elle se lance en choisissant "Affichage/Vue géographique" dans le menu principal.
Vous pouvez modifier les marqueurs sur la carte ou filtrer les lieux à afficher. Vous pouvez également afficher la liste des lieux depuis cette vue.


Utilisation de la liste des lieux

L'utilisation de la liste des lieux est très simple. Elle se lance en choisissant "Affichage/Liste des lieux" dans le menu principal.
La liste des lieux est classée par nom de commune croissant, quelque soit la façon dont vous avez ordonné les juridictions.

Les lieux dont les coordonnées ne sont pas trouvés sont en rouge. Quand on déplie un lieu, les événements généalogiques s'y rapportant apparaissent en arborescence. Un cli droit sur un lieu ou sur un événement fait apparaitre un menu déroulant.
Vous trouverez dans cette aide les explications concernant le menu lieu et le menu événement.
