Migração de outro programa


Já tem uma genealogia criada com outro programa e acabou de decidir passar a utilizar o Ancestris.

A migração precisa de duas etapas:

  • 1/ Preparação: a partir do seu programa anterior, tem de exportar os dados para um ficheiro de formato GedCom (extensão .ged).
    Todos os bons programas permitem esta operação. Se encontrar problemas (muitas vezes devido ao não cumprimento da norma Gedcom) consulte o capítulo seguinte Problemas identificados durante a importação.
  • 2/ Importe os seus dados para o Ancestris:
    • A partir de Ficheiro/Abrir, escolha o seu ficheiro xxxx.ged;
    • selecione-o e clique em Aceitar.


Antes de qualquer outra coisa, deve agora verificar a Ordem das jurisdições.
