Migrating from another software


You already have a genealogy created with another software and just decided to start using Ancestris.

The migration takes two steps:

  • 1/ Preparation: from your previous software, you must export your data to a file in Gedcom format (extention .ged).
    Every good software allows this operation. Should you find any problems (mostly due to non-compliance with the Gedcom standard) see the next chapter Identified problems during import.
  • 2/ Import you data to Ancestris:
    • From menu "File/Open", choose you xxxx.ged file;
    • select it and click OK.


Now, before anything else, you should check the Order of jurisdictions.
