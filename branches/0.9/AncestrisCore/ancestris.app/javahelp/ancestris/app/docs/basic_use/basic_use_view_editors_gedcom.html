L'Éditeur GedCom


Cet éditeur vous permet d'accéder à la pleine puissance de la grammaire/syntaxe GedCom.

Vous allez y faire toutes les manipulations possibles de façon très précise sur l'entité (personne, famille, etc) en cours.

Les propriétés sont disposées de manière :arborescence:hiérarchisées: selon la norme gedcom.
Un symbole est affiché devant chaque tag/etiquette gedcom puis le nom de l'étiquette et la valeur.
Si ce symbole est un point d'interrogation: tag inconnu ou mal placé.

Description :
Partie haute : affichage les différentes propriétés (les sous-propriétés peuvent être masquées/affichées par clic sur les poignées [+] / [-]),
Partie basse : tags/etiquettes + valeurs éditables à cet endroit.

Ajouter une propriété :
L'ajout de la propriété se fait par un clic droit sur la première ligne de la fenêtre du haut et choisir la seconde ligne 'Ajouter une propriété' (Si vous ne trouvez pas ce que vous chercher prenez la première ligne).
Si dans aucune des deux solutions d'Ajout de propriétés vous ne voyez pas celle recherchée, c'est que très certainement elle n'est pas autorisée à cet endroit.
Certaines sous-propriétés sont automatiquement ajoutées (les plus courantes).
Si vous avez besoin de sous-propriété il vous faut refaire la manipulation en cliquant droit sur la propriété concernée.
Une fois que vous avez ajouté les propriétés vous devez les remplir en remplissant les champs adéquats dans la fenêtre du bas (positionnez-vous sur l'élément à remplir dans la fenêtre du haut et remplissez dans celle du bas).

N'oubliez pas de valider vos saisies/changements.
