Author of this Genealogy - The Submitter

Information requested in this screen are mandatory for all genealogy files. You must fill them up carefully.

Full Name

Here you must write the first name and last name.
  • Example: John JOHNSON
  • For married women, here you are free to write what you want. For example, if your last name is KENNEDY, and your husband is BRADBURY, you can write: Belinda KENNEDY, or Belinda BRADBURY-KENNEDY, or Belinda BRADBURY.

Address

The address must be written in a postal way.
  • Example: P.O. Box. 317
    455 Larkspur Dr.
    Apartment 123

Zip Code

Or Postal Code. No need for more explanation.

City

Please be careful with the way you write the city name. If a city name has "-", don't forget them.
  • Example: Saint-Louis

State

In different countries, you can have states. In U.S.A, Texas, New Mexico, etc. are states..
  • Example: New Mexico (or NM)

Country

No need for more explanation.
  • Example: United States of America (or U.S.A.)
