File name prefixes

q / s
The type of rules that at least should have been applied with the last conversion.
FAM / INDI / ...
Launch the report from the type of entity.
Known bug: you may first have to execute another type of query.
SOSA / TREETOP
Custom properties generated with the report "tool box".

Files

Copy the examples on your local file system. For details on customisation see comment inside and the general description of queries and rules. Files with rules can't have comment.

rules

Applied to the converted gedcom to provide shortcuts for the queries.

  • QuickRules.txt
  • SlowRules.txt QuickRules + ancestor/descendant relations.
