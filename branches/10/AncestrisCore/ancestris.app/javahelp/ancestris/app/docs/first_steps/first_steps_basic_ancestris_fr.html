Les premiers paramètres


Vous venez d'installer Ancestris. Des options par défaut vous permettent de commencer immédiatement la saisie de vos premières données. Cependant, il est recommandé de rapidement vous familiariser avec certains paramétrages afin d'adapter l'environnement de votre logiciel à votre façon de travailler et de donner à votre généalogie l'apparence souhaitée.


Cette personnalisation se fait dans le menu Préférences, accessible à partir de:
  • la barre menu /Outils/Préférences,
  • l'icône ''Préférences'' (bord droit de la barre outils),
  • la fenêtre de démarrage* /Mon Ancestris/Configuration d'Ancestris.
    (* visible au lancement d'Ancestris puis à chaque démarrage si la case (en bas) est cochée.)

De là, vous pourrez:
  • dans la fenêtre ''Apparence'':
    • choisir le mode de gestion des fenêtres,
    • choisir entre deux modes de saisie des juridictions, (option "Distinguer les lieux lors de l'édition des juridictions)

      • ou

    • modifier l'apparence des fenêtres (il faudra redémarrer pour valider),
    • augmenter le nombre maximum d'annulations.
  • dans la fenêtre ''Affichage'':
    • décider, entre 4 possibilités, du format des dates,
    •   pour le paramètre ''Textes'' il est FORTEMENT DÉCONSEILLÉ de modifier le paramètre par défaut -246-,
    • indiquer vos préférences à l'accès de vos données dans l'onglet ''Données privées''.
  • dans la fenêtre ''Données'':
    • choisir si, au moment de créer l'ascendance d'un individu, un seul ou les deux parents doivent être créés,
    • choisir quelles juridictions vous souhaitez saisir et opter pour leur ordre d'affichage (voir Gestion des lieux),
    • opter pour l'affichage des patronymes en majuscules (recommandé pour éviter toutes confusions avec les prénoms),
    • pour ''Encodage'', même si ce n'est pas la norme Gedcom, il est préférable d'indiquer ''UTF-8'' (voir Utilisation avancée).
  • dans la fenêtre ''Fichier'':
    • indiquer l'adresse de votre fichier Gedcom.

Chaque option est accompagnée par une aide sous forme d'info-bulle.

Ceci n'est qu'une sélection de l'ensemble des préférences offertes. Testez, essayez... vous pourrez toujours revenir sur vos choix.
