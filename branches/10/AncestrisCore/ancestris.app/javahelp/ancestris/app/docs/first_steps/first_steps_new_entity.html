Nom :

Nom de famille : le patronyme de l'individu
Prénoms : le ou les prénoms de l'individu
Puis, cliquer sur "OK", en bas à droite de la fenêtre de l'éditeur.

Astuce : Si vous avez saisi plusieurs prénoms séparés par des espaces, vous verrez apparaître après validation dans la ligne Prénoms chacun des prénoms séparés par une virgule, ceci est tout à fait normal, Ancestris réécrit cette ligne pour la mettre "à la norme Gedcom". Vous verrez que le nom de votre individu est "correctement" écrit, sans virgule, dans l'arbre comme dans l'intitulé de l'éditeur Ancestris
Pour aller plus loin vous pouvez paramétrer les espaces dans préférences/Données/Données/Nom.

Astuce : Pour saisir les variantes du nom d'un individus voir : FAQ : Un individu est connu sous plusieurs patronymes

Au préalable il est recommandé d'adopter un ordre des juridictions universel et homogène !

Quelques conseils pour l'ordonnancement des juridictions.
Pour saisir un lieu : saisir les juridictions par ordre croissant en les séparant par une virgule. Si un niveau est inconnu, marquer son emplacement par une virgule.

Cliquer sur "OK" pour valider vos ajouts.

Astuce : Ancestris vous propose d'utiliser 7 niveaux de juridictions :
Lieudit,Commune,Code_INSEE,Code_Postal,Département,Région,Pays
Ainsi vous augmentez vos chances de géolocaliser votre lieu sur la carte.

Astuce : Pour renseigner un lieu vous pouvez séparer les juridictions en plusieurs champs dans Préférences / Apparence / Edition
