Keyboard Shortcuts

GenealogyJ supports keyboard shortcuts. Which shortcuts are available depends on the currently active view:

Keyboard ShortcutAssociated action
In Control Center
ctrl + n create new file
ctrl + o open file
ctrl + s save selected file
ctrl + x exit GenJ
In all Views
ctrl + z undo
ctrl + y redo
ctrl + s save file
ctrl + w close view
ctrl + e jump to Edit View
ctrl + f jump to Family Tree View
ctrl + n jump to Navigator View
ctrl + r jump to Report View
ctrl + s jump to Search View
ctrl + t jump to Table View
ctrl + spaceopen context menu
In Table View
ctrl + rightswitch to next entity type (e.g. from individuals to families)
ctrl + left switch to previous entity type (e.g. from families to individuals)
In Edit View (Gedcom mode)
ctrl + x cut selected node(s) to clipboard
ctrl + c copy selected node(s) to clipboard
ctrl + v paste clipboard into selected node
