Views - Edit View

This view allows the user to edit the values associated with an entity (or record) within GenJ. This could be a place or a date pertaining to a person. It also could be the name of an external file. All the displayed fields can receive data and all changes have to be committed by clicking on 'OK' at the bottom. To cancel and revert back to the current state click on 'Cancel'.

There's an underlying structure to every entity in Gedcom that can be made visible by clicking on the mode-button. This might be necessary at times to investigate the hierarchy of (extended) properties and their values. In this mode any selection of a node in the hierarchy at the top is followed with an editor at the bottom for that specific Gedcom property. Every displayed node at the top will also show a Context Menu if clicked with the right mouse-button.

Toolbar

  • go back to the last entity edited
  • pin the editor to the currently edited entity (won't follow focus)
  • undo the last set of changes committed with 'OK'
  • redo the last set of changes undone with undo
  • perform actions on the currently edited entity (e.g. add a spouse or child to a person)
  • switch between 'Standard' and 'Gedcom' mode

    Settings

    There are no settings for this view at this point.
