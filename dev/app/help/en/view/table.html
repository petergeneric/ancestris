Views - Table View

This view shows entities in rows - the columns are properties of those entities (e.g. Name, Sex and Birth Date). You can scroll up and down browsing the rows with the cursor-keys or the scroll-bar on the right. To sort the rows by a specific column simply click the appropriate column-header. A small marker will indicate the column and direction sorted by.

Now how are the cell values calculated? The table only shows one type of entity at a time. Each row maps to one instance of the type currently shown. The cells in each row are property-values of an entity identified by Tag Paths. The path for a column is displayed in the header (e.g. INDI:BIRT:DATE is the date of birth of the individual contained in a row).

Settings

The settings for this view are grouped by type of entity. For each type (select at the top) the columns to be shown are marked in the center (un/check nodes in the center). The order of columns from left to right is shown at the bottom (use buttons on left to reorder).
