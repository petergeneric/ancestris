GenjFr - Nomade


Vous avez envie de vous balader avec toute votre généalogie avec vous, pour la montrer par exemple à un copain qui a un ordinateur, etc... GenjFr a été prévu pour ça également. Comment faire?
  • On commence par repérer où est sa clé USB dans l'arborescence de vos répertoires.
    Suivant que vous êtes sous Windows ou Linux, vous devriez voir apparaître votre clé dans votre navigateur de fichiers.
  • Vous créez sur la racine de votre clé, un répertoire que l'on pourrait appeler "genjfr".
    Sous Windows, vous pourriez avoir quelque chose du genre : E:\genjfr (si E: est la lettre attribuée à Windows à votre clé), sous Linux, on aurait par exemple : /media/mykey/genjfr (mykey étant le nom de ma clé).
  • Dans ce sous répertoire "genjfr", vous placez le fichier que vous venez de télécharger. Il s'appelle en général genjfrapp_xxxxx.zip. (xxxx est à remplacer par le numéro de version du fichier que vous venez de télécharger - ex: genjfrapp_0.2-2.zip ou genjfrapp-20100226.zip)
  • Vous décompressez le fichier genjfrapp_xxxxx.zip, il va se créer tous les sous-répertoires nécessaires. Une fois que c'est fait, vous pouvez pour gagner un peu de place, effacer le fichier genjfrapp_xxxxx.zip, vous n'en aurez plus besoin.
  • Si vous êtes sous Windows, vous pourriez utilement mettre dans le répertoire "genjfrapp\bin" de votre clé (c'est là que vous allez trouver un fichier exécutable, qui s'appelle genjfrapp.exe), un petit fichier bat pour lancer genjfr. Sous Windows, un fichier bat est un fichier de commande exécutable). Pour vous simplifier la vie, vous ouvrez un éditeur texte, genre Notepad, et vous faites un copier/coller, du texte ci-dessous (les 3 lignes qui suivent et attention qu'elles soient bien sur trois lignes) dans Notepad, ensuite vous sauvegardez ce texte en appelant le fichier nouvellement créé "rungenjfr.bat" (faites "Sauvegarder Sous").
    rem - Ce fichier sert a lancer le logiciel de genealogie GenjFr -
    @echo off
    genjfrapp.exe --userdir ..\.genjfrapp
    Voilà un exemple de ce à quoi ressemble l'arborescence des fichiers de la clé USB sous un système Windows :

  • Je vous conseille de créer un sous-répertoire genjfr\gedcom (sous Windows), ou genjfr/gedcom (sous Linux), et dans ce sous-répertoire "gedcom", vous y mettez votre ou vos fichiers gedcom.
  • Vous pouvez également créer un sous-répertoire genjfr\documents (sous Windows), ou genjfr/documents (sous Linux) dans lequel seront stockés les documents crées.
  • Vous êtes sous Windows, alors il vous suffit de cliquer sur rungenjfr.bat, il va créer automatiquement un répertoire .genjfrapp juste à coté du dossier genjfrapp.
  • Si déjà vous avez votre configuration aux petits oignons, pourquoi ne pas récupérer tout votre sous répertoire ".genjfrapp" et installez le sur votre clé. Vous pouvez aussi utiliser la fonction d'importation des paramètres de GenjFr si vous avez pris la précaution sur votre ordinateur principal, de faire un export vers un fichier de configuration zippé de votre choix. Lors du lancement à partir de la clé USB, si vous avez copié dessus, votre fichier de configuration zippé, vous allez dans les paramètres de GenjFr, et vous faites l'import en donnant le chemin et le nom du fichier qui se trouve sur votre clé.
    Vous reprendrez ainsi tous les paramétrages, ainsi que tous les fichiers nécessaires à la localisation (visualisation géographique).

Windows :
Pour lancer maintenant GenjFr vous ne cliquez plus sur l'exécutable mais sur «rungenjfr.bat» que vous venez d'installer, vous pouvez également créer un raccourci que vous placez à la racine de votre clef USB.
