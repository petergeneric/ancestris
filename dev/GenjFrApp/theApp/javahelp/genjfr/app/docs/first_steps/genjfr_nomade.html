GenjFr - Nomade


Vous avez envie de vous balader avec toute votre généalogie avec vous, pour la montrer par exemple à un copain qui a un ordinateur, etc... GenjFr a été prévu pour ça également. Comment faire?
  • Il suffit de mettre son fichier gedcom sur une clé USB et d'installer les fichiers suivants sur cette même clé:
    • "genjfrapp"
    • "Run.Bat" à créer et placer dans le même répertoire que l'exécutable genjFrapp.exe.
      Vous saisissez le texte suivant dans le fichier run.bat :
      genjfrapp.exe --userdir ..\.genjfrapp
  • Au niveau de l'organisation, on pourrait par exemple tout installer sur la clé dans un répertoire "genjFr".
  • je vous conseille de créer un sous-répertoire genjfr\gedcom, et dans ce sous-répertoire gedcom, vous y mettez votre ou vos fichiers gedcom.
  • Vous pouvez également créer un sous-répertoire genjfr\documents dans lequel seront stockés les documents crées.
  • Dans votre répertoire GenjFr vous décompressez genjfrapp
  • Vous placez run.bat dans le répertoire où vous avez l’exécutable genjfrapp.exe
  • Vous cliquez sur run.bat, il va créer automatiquement un répertoire .genjfrapp juste à coté du dossier genjfrapp.
  • Si déjà vous avez votre configuration aux petits oignons, pourquoi ne pas récupérer tout votre sous répertoire ".genjfrapp" et installez le sur votre clé.

Vous reprendrez ainsi tous les paramétrages, ainsi que tous les fichiers nécessaires à la localisation (visualisation géographique).

  • Pour lancer maintenant genjFrapp vous ne cliquez plus sur l'exécutable mais sur run.bat que vous venez d'installer, vous pouvez également créer un raccourci que vous placez à la racine de votre clef USB.
