La vue Entité


Cette vue permet de visualiser la liste des propriétés de l'entité sélectionnée dans une des autres fenêtres :
  • Editeurs
  • Table
  • Arbre
  • Navigateur
  • Chronologie
Cette vue ne permet pas de naviguer.
Les informations affichées peuvent être personnalisées : calques. Les calques définis dans cette vue peuvent être beaucoup plus poussés et détaillés que dans la vue Arbre.
(dans la vue arbre, cela demanderait beaucoup ressources vu le grand nombre d'individus et de familles à afficher.)

Cette vue permet d'utiliser le menu contextuel
Ce menu est disponible par clic droit.

Liste des différents outils disponibles dans le menu à droite:
Toutes les modifications apportées à cette liste peuvent être validées (bouton "appliquer") ou abandonnées (bouton "rétablir" ou directement "fermer").
  • Réglage des paramêtres :
    • Onglet "généraux" :
      • Antialiasing : Affichage normal ou lissage
    • Onglet "structures des cadres" :
      • Pour chaque type d'entité, vous pouvez sélectionner un des calques proposés par défaut ou en créer un.
