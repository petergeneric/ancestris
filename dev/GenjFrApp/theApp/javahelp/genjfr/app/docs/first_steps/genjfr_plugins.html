GenjFr - Les Plugins


Les "Plugins" (ou plug-in, ou module ou greffon ou plugiciel) sont des programmes, qui viennent s'ajouter au programme principal (donc à GenjFr) pour lui donner de nouvelles fonctionnalités.
Un plugin ne peut pas fonctionner seul, il lui faut le programme principal.

Les plugins permettent aussi aux personnes qui veulent se lancer dans la programmation java, de mettre au point des applications plus ou moins complexes, qui ne risquent pas de géner l'application principale, mais qui vont donner aux utilisateurs de nouvelles possibilités.

Plusieurs questions peuvent se poser :
  1. Comment savoir si des plugins sont disponibles?,
  2. Le téléchargement et l'installation des plugins,
  3. Désinstaller un plugin,
  4. Ecrire un plugin,

I - Comment savoir si des plugins sont disponibles?


Pour savoir si de nouveaux plugins sont disponibles, il faut aller dans le menu "Outils" - "Plug-ins". Une fois la fenêtre ouverte, vous cliquez sur le deuxième onglet "Plug-ins disponibles (x)" (x est le nombre de plugins nouveaux que le logiciel a trouvé).

Dans l'onglet "Plug-ins disponibles (x)", vous cliquez sur "Recharger le catalogue". Une liste va s'afficher ou se mettre à jour.
En cliquant sur un des plugins qui apparaît, vous aurez dans la fenêtre de droite des informations complémentaires sur ce plugin.
Si vous voulez l'installer, vous cochez la case en face de son nom.

II - Téléchargement et installation


Vous venez de décider d'installer un ou plusieurs plugins. Vous allez voir en bas à gauche un bouton devenir actif dénommé "Installer". Cliquez dessus.
Une nouvelle fenêtre s'ouvre, c'est l'installateur de plugins. Il vous résume les plugins qui vont être associés à GenjFr. Ca vous convient, alors cliquez sur "Suivant". La fenêtre qui apparaît vous donne les conditions pour pouvoir utiliser ce plugin, c'est l'aspect légal des choses par l'acceptation de la licence d'utilisation. A priori vous allez être d'accord, donc vous allez cocher la petite case en bas "J'accepte tous les termes du contrat de licence".
Une fois cette case cochée, le bouton "Installer", devient actif. Cliquez sur "Installer".
Le plugin est téléchargé, et une fenêtre apparaît qui vous dit si oui ou non le plugin est "signé". Quelque soit la présence ou non de cette signature, aucun risque, ce plugin vient bien du site officiel de GenjFr. Cliquez sur le bouton "Continuer".
Et voilà, l'installation est terminée. En fonction du plugin installé, il est possible que le programme vous dise qu'il va redémarrer le programme pour pouvoir prendre en considération ce nouveau plugin.

Regardez maintenant dans le menu "Options". Un ou plusieurs nouveaux plugins sont apparus. Ce sont ceux nouvellement installés.

III - Désinstaller un plugin


Vous ne voulez plus utiliser un plugin, soit il ne vous plait pas, soit il fait double emploi avec un autre plugin, soit encore vous avez simplement voulu le tester, mais vous ne voulez pas le conserver.

Vous allez dans "Options", "Plug-ins". Le menu des plugins apparaît. Là vous allez dans le quatrième onglet "Installés (x)" (x étant le nombre de plugins installés sur votre système).
Cochez la case en face du plugin que vous voulez installer, et vous cliquez après sur le bouton "Désinstaller", qui est devenu actif.
Un menu vient vous résumer quels sont les plugins qui vont être désinstallés. Cliquez si vous êtes d'accord avec ce qui vous est proposé, sur "Désinstaller".
Après cette suppression, un menu apparaît vous demandant si vous voulez ou nous redémarrer immédiatement ou plus tard, GenjFr. Il faut mieux redémarrer tout de suite. Votre choix étant fait, cliquez sur "Terminer".
GenjFr redémarre et vous pourrez constater dans le menu "Options", que le ou les plugins nouvellement supprimés, ne sont plus là.
Facile non!
