GenjFr - Les Plugins


GenjFr vient avec des modules, ou greffons, ou plugins (tous ces mots visant la même chose), pré-installés, mais il vous permet aussi d'en télécharger de nouveaux.

Voici quelques plugins qu'il vous est possible de reprendre si vous ne les avez pas déjà.

  1. La Calculette

    Cet outil vous permet comme n'importe quelle calculatrice de faire différentes opérations, genre addition, soustraction, division, et multiplication, etc.

    Elle est également un "convertisseur de dates". Une date rentrée en calendrier grégorien par exemple, est convertie en différents formats, dont le calendrier républicain.
    Pour ce faire, vous entrez votre date, exemple 14 juillet 1795. Vous cliquez sur la petite icone qui se trouve à gauche de la date, et vous allez voir apparaître votre date convertie au calendrier julien, hébreu et républicain. Ainsi pour le 14 juillet 1795 nous étions le sextidi 26 Messidor An III.
    Si vous voulez convertir une date d'un calendrier républicain en calendrier grégorien, commencer par cliquer sur l'icone à gauche des champs de dates. Vous choisissez "Calendrier Républicain". Ensuite vous entrez votre date au format calendrier républicain, exemple, 25 Germinal An II, et en recliquant sur la même petite icône que ci-dessus, vous voyez votre date convertie dans les trois autres calendriers, dont le calendrier grégorien. Nous étions le lundi 14 avril 1794.

  2. Feedback

    Ce module que l'on pourrait appeler "Contact avec l'équipe de GenjFr", vous permet de prendre contact avec l'équipe de développement de GenjFr si vous avez envie de leur mettre un petit mot, leur faire savoir que le logiciel vous satisfait ou pas, etc, mais aussi si vous avez des problèmes avec l'application qui nécessiteraient l'envoi d'éléments de configuration de votre système, ainsi que du log (fichier journal) de GenjFr.
    Quelques paramètres sont nécessaires :
    • Celui qui est obligatoire : Serveur de mail sortant :
      C'est le serveur smtp de votre fournisseur d'accès, celui que vous mettez dans votre logiciel de messagerie. Si vous êtes chez Free par exemple, ce serait "smtp.free.fr". Voilà quelques serveurs smtp les plus couramment utilisés.
    • Celui qui est nécessaire : Votre adresse mail :
      Si vous voulez qu'on vous réponde, il faut qu'on ait votre adresse Email. A priori vous savez ce que c'est. Exemple : jean.emile@wanadoo.fr
    • Votre nom :
      C'est mieux de pouvoir vous appeler de votre nom, plutôt que de dire : bonjour machin.
    • Sujet :
      Là encore, il faut mieux mettre le sujet de votre message. S'agit-il d'un problème, d'un remerciement, d'une critique, etc.
    • Envoyer le fichier journal :
      C'est le fichier log de GenjFr. Surtout laissez cochée cette option si vous voulez nous rapporter un problème. Si au contraire, vous voulez nous dire tout le bien que vous pensez de GenjFr, là pas besoin de joindre le log.
    • Destinataire :
      Ne changez pas cette adresse, sans quoi votre message ne nous parviendra pas.
    • Commentaire :
      Là à priori, c'est après cette ligne que vous écrirez votre message. Laissez les lignes qui se trouvent au dessus, ça donne sous quel environnement vous êtes.
    Une fois votre message écrit, une petite fenêtre s'ouvrira pour vous remercier.
