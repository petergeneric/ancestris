Mise en oeuvre des calques


Les calques sont des fichiers au format html, donc le format que l'on utilise pour créer des sites internet, qui vont vous permettre de donner un tout autre aspect à certaines des vues de GenjFr.

Quelles vues sont concernées par les calques?


A tout seigneur, tous honneurs, c'est le fenêtre de l'arbre généalogique. Grâce à différents calques, vous allez pouvoir vous faire un arbre avec un aspect très dépouillé, en y mettant par exemple que les nom et prénoms, dates et lieux de naissance et décès. A l'inverse, vous allez à l'aide d'un autre calque, pouvoir mettre un maximum d'informations pour chaque personne, avec outre les renseignements élémentaires, mais aussi la profession, les adresses, la date d'un éventuel divorce, etc, etc.

Autre vue : La vue Entité.
Là encore, suivant le calque que vous utilisez, vous allez pouvoir avoir une véritable fiche individuelle ou familiale.

La dernière vue, et c'est surement la moins connue, c'est l'éditeur gedcom.
Dans la partie basse de l'éditeur, il est possible d'afficher plus ou moins de choses. Vous le comprendrez mieux en regardant les captures d'écran partielles ci-dessous.

Comment changer de calque?


Lorsque GenjFr est lancé, vous allez dans la fenêtre de l'arbre généalogique par exemple, et vous ouvrez le menu des paramètres de l'arbre (c'est la petite icone à côté de celle représentant une imprimante).
Vous allez dans le dernier onglet dénommé "Structure des cadres". Dans la fenêtre qui apparaît, vous avez à gauche chacune des entités (individus, familles, etc) pour laquelle des calques sont possibles. Cliquez sur le petit triangle à gauche de chacune d'elles et vous allez voir les différents calques possibles ; choisissez celui que vous voulez, et faites "Appliquer". Vous verrez immédiatement le résultat sur votre arbre généalogique.

Où trouve t'on les calques?


Les calques sont situés dans le répertoire des paramètres de GenjFr, dans un sous-répertoire dénommé "blueprints".

Sous Windows le répertoire est ici :
à compléter
Sous Linux vous trouverez ce sous-répertoire "blueprints" ici :
"/home/votre_repertoire/.genjfrapp/dev/genjfr/blueprints"

Ce sous-répertoire "blueprints", comporte lui-même plusieurs sous-répertoires :
      INDI
      FAM
      OBJE
      NOTE
      SOUR
      SUBM
      REPO

C'est dans chacun de ces sous-répertoires que l'on va trouver les différents calques.
Dans INDI, par exemple, vous trouverez les calques concernant la représentation des INDIvidus. Dans le répertoire FAM, vous aurez les calques pour la représantation des familles, etc... OBJE est relatif aux objets multimédias, NOTE, concerne les notes, SOUR les sources, SUBM, les personnes ayant fourni des informations, et enfin REPO la représentation des lieux de dépôts.

Le calque lui-même que vous allez trouver dans ces répertoires est un fichier ayant l'extension .html ; ils sont appelés en général du nom de la personne qui l'a fait, suivi de son extension. Exemple : marcel.dupont.html.
Les calques émanant du même auteur, s'appellent tous de la même manière, mais comme ils sont situés chacun dans son sous-répertoire, il n'y a aucun problème.

Dans cet exemple, pour chacune des entités (INDI, FAM, etc), vous avez trois calques, que vous pourrez choisir à votre gré.

Comment avoir de nouveaux calques?


En les faisant soi-même
Plusieurs outils existent sur l'internet pour créer des pages html. Recherchez un éditeur html, et lancez vous dans la création. N'oubliez pas de partager votre résultat avec tous les autres utilisateurs de GenjFr. Le site web de GenjFr vous donne toutes les indications pour transmettre vos calques aux autres.

En les reprenant sur le site d'échange
En allant sur le site de GenjFr, vous allez pouvoir reprendre un paquet reprenant l'ensemble des calques que des utilisateurs ont accepté de partager avec vous.
Une fois le fichier téléchargé, vous lancez GenjFr. Vous ouvrez "Outils" - "Préférences", et là vous faites "Importer" (c'est le bouton que vous allez trouver en bas à gauche du menu de configuration). Dans la fenêtre qui s'ouvre, vous utilisez le petit navigateur (Source de l'import) en haut pour mettre le nom du fichier que vous venez de télécharger (blueprints_archive.zip). Ensuite dans "Sélectionnez les options pour l'import", vous cochez juste "Calques personnalisés" et voilà, tous les blueprints sont créés là où il faut.
