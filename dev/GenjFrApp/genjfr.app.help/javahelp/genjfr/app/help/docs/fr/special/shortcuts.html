Les Raccourcis Clavier

GenJ supporte les raccourcis clavier.
Savoir quels raccourcis sont possibles, dépend bien entendu de la fenêtre actuellement active :


Raccourci Clavier Action Associée
Au niveau du Centre de Contrôle
ctrl + ncréation d'un nouveau fichier gedcom
ctrl + oouvrir un fichier gedcom
ctrl + ssauvegarder le fichier gedcom sélectionné
ctrl + xquitter GenJ
Dans toutes les Vues
ctrl + zdéfaire
ctrl + yrefaire
ctrl + ssauvegarder le fichier
ctrl + wfermer la fenêtre
ctrl + epasser sur la fenêtre de l'Editeur
ctrl + tpasser sur la fenêtre du Tableau des Données
...passer vers ...
ctrl + Espaceouvrir le menu contextuel
Dans le Tableau des Données
ctrl + flèche droitebasculer vers l'entité suivante (ex: des Individus vers les Familles)
ctrl + flèche gauchebasculer vers l'entité suivante (ex: des Familles vers les Individus)
Dans l'Editeur (Editeur Gedcom)
ctrl + xcouper et envoyer le ou les node(s) sélectionné(s) vers le presse-papiers
ctrl + ccopier le ou les node(s) sélectionné(s) vers le presse-papiers
ctrl + vcoller le presse-papiers vers le node sélectionné
