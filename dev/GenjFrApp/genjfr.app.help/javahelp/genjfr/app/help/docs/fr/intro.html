GenJ est un programme permettant de visualiser et d'éditer des données généalogiques, écrit en langage Java. Vous êtes fortement invité à l'utiliser, et ce tout à fait gratuitement, à télécharger le code source et/ou vous joindre à cette équipe et contribuer à son développement que ce soit par l'écriture du programme ou tout simplement en aidant à sa traduction.

Cette aide en ligne vous donne un point de départ vers les fonctionnalités de GenJ et vous décrit un peu son fonctionnement interne et ses spécificités cachées. Disposer d'un éditeur puissant est agréable mais certaines choses parmi toutes les possibilités ne sont pas faciles à trouver pour l'utilisateur débutant. Veuillez lire les sections sur les Concepts généraux dans un premier temps, et vous disposerez par la suite des connaissances nécessaires pour travailler avec les différentes Vues qui les mettent en oeuvre. Les FAQ (questions fréquemment posées) vous fourniront les astuces pour travailler avec GenJ.

Ce document vous donne également les points de départ avec GenJ, et notamment toutes les méthodes pour installer sur votre ordinateur la machine virtuelle Java (sans laquelle pas de salut, genj ne pourrait pas fonctionner), mais aussi GenJ lui-même, et ce sous trois systèmes d'exploitation : Windows, Linux et Mac. Enfin un petit tutorial a été fait pour débuter avec GenJ (vos premiers avec le logiciel).

GenealogyJ (GenJ) est un logiciel libre ; vous pouvez le redistribuer et/ou le modifier en respectant les termes de la licence GNU General Public Licence telle qu'elle est publiée par la Free Software Foundation.

En ce qui concerne les derniers développements, et/ou si vous êtes intéressé par ce projet et désirez aider, vous pouvez visiter http://genj.sourceforge.net.

Bonne Lecture !
