F Um was geht es bei GenJ?
A GenealogyJ (kurz GenJ) ist ein Anzeige- und Bearbeitungsprogramm für genealogische Daten im Gedcom-Format.

F Gibt es ein Diskussionsforum bezüglich GenJ ?
A Ja, bitte gehen Sie zu unserem Forum @ genj.sourceforge.net

F Was bedeutet Gedcom?
A Bitte lesen Sie die Erklärung hier.

F Wie kann ich Bilder in der Baum-Ansicht anzeigen lassen?
A Um Bilder für jede Person in der Baum-Ansicht anzuzeigen, fügen Sie jeder Person eine Multimedia-Eigenschaft (OBJE) hinzu. Lassen Sie eine Person in der Editier-Ansicht anzeigen, wählen Sie die in der Hierarchie oberste Eigenschaft INDI aus, klicken Sie auf den '+' Auswahlknopf und wählen Sie OBJE zum Hinzufügen aus. Lassen Sie die Untereigenschaft FILE auf eine Datei in Ihrem Computer-Dateisystem verweisen (Auswahlknopf '>>') und die Bilder werden aus der Baum-Ansicht heraus angezeigt werden (Die Ansicht sucht automatisch nach dem INDI:OBJE:FILE).

F Kann ich die von GenJ benutzte Sprache ändern?
A Bitte laden Sie das passende Sprach-Paket (genj_ll-x.y.zip) für GenJ (z.B. genj_fr-2.0.zip) von der Projekt-Homepage herunter und folgen Sie den Anweisungen, um es zu installieren. GenJ wird automatisch versuchen, die Sprache Ihres Betriebssystems zu verwenden. Um die verwendete Sprache explizit zu ändern (das Sprach-Paket muss manuell installiert werden), wählen Sie den Menüeintrag Hilfe|Über und den Menüpunkt Look&Feel. Nun wählen Sie die gewünschte Sprache aus.

F Wie kann ich einen Zentimeter maßstabsgetreu auf meinem Bildschirm darstellen?
A GenJ benutzt Maße der realen Welt für verschiedene Komponenten (z.B. in der Baum-Ansicht). Um die Längen von Linien und Größen von Boxen korrekt auf Ihrem Bildschirm anzuzeigen, müssen Sie GenJ Ihre Bildschirmauflösung mitteilen. Um die Auflösung explizit anzupassen, wählen Sie den Menüeintrag Hilfe|Über und dann den Menüpunkt Look&Feel. Benutzen Sie ein Lineal, um das Lineal auf dem Bildschirm entsprechend anzupassen.

F Wie Kann ich die Darstellung von GenJ ändern?
AGenJ verwendet das Swing-Toolkit zur Darstellung von Informationen und benutzt standardmäßig einen dem System angemessenen Look. GenJ erlaubt auch die Änderung des Look&Feel. Bitte laden Sie das Look&Feel-Paket (genj_lnf-x.y.zip) von der Projekt-Homepage und folgen Sie den Installationsanweisungen. Nach dem Neustart von GenJ kann der Look&Feel in der Liste der Optionen geändert werden.

F Ich habe eine vorhandene Person und möchte sie als Kind einer anderen schon existierenden Person einfügen. Die einzige verfügbare Option scheint das Hinzufügen einer *neuen* Kind/Eltern-Beziehung zu einem existierende Datensatz zu sein, aber man kann scheinbar keine *existierende* Person einer Kind/Eltern Beziehung hinzufügen. Wie kann ich das tun?
A Sie können den existierenden Elternteil (oder die Eltern-Familie des 'existierenden' Kindes) auswählen. Wählen Sie dann "Kind hinzufügen" aus dem Kontextmenü und in dem angezeigten Dialog wählen Sie oben die gewünschte Person in der DropDown-Auswahl. Auf diese Weise wird keine neue Person als Kind angelegt sondern nur eine Beziehung zwischen den schon existierenden erzeugt.

F Warum werden Bilder in unterschiedlichen Größen dargestellt?
A Bei der Aufarbeitung der Ausgabe spielen Vorlagen eine Rolle. Abhängig davon, wieviel in einem Kasten angezeigt werden muss, kann der Platz für ein Bild begrenzt sein. Es wird dann verkleinert, um genügend Platz für Text zu haben. Wenn genug Platz vorhanden ist, so wird die Original-Bildgröße verwendet (abhängig von Größe *und* Auflösung). Darum kann die Darstellung variieren.

F Wie kann ich Information über eine Adoption abbilden?
A Füge in der Editier Ansicht der adoptierten Person ein Ereignis ADOPtion hinzu. Dann ist es möglich durch das Kontextmenü eine Referenz zur adoptierenden Familie festzulegen.
