Les Entités - Lieux de Stockage

Comme une Source et un Fournisseur d'information ce type d'entité enregistre la trace de l'endroit d'où viennent les informations généalogiques. Un lieu de stockage est une sorte d'enregistrement pour conserver une entité (voir les infos ci-dessous).
