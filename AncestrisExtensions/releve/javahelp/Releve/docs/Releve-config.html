Configuration  

Les paramètres de configuration sont rassemblés dans l'onglet Configuration présent à coté des onglets de tables 



La barre d'outil

La barre d'outils contient les commandes de gestion des fichiers. Voir le détail des commandes dans la rubrique Les fichiers.
Elle contient aussi le bouton Relevé démo qui permet d'afficher un relevé avec quelques dizaines d'évènements pour tester l'outil.

Commune

Cette rubrique sert à indiquer la commune de tous les évènements présents dans le relevé.
Le nom de la Commune  est obligatoire
Les autres informations Département, Région , Pays  sont facultatives.

Remarque : ces informations ne sont utilisés qu'au moment d'enregistrer les données dans un fichier, ou quand on importe des données supplémentaires d'un autre fichier.

Statistiques

Les statistiques indiquent le nombre de relevés de chaque types présents en mémoire.

Options 

Ces options permettent d'autoriser ou non les fonctions automatisées de l'éditeur .

Signaler les nouveaux noms , prénoms et professions

Quand cette option est activée, un message d'avertissement est affiché si un nom ou prénom ou profession n'existe pas déjà dans les relevés.
Cette avertissement permet en particulier d'éviter les fautes de frappe sur des termes couramment utilisés.

Signaler les doublons

Quand cette option est activée, un message d'avertissement est affiché si un même relevé existe déjà avec une date d'évènement , un nom et un prénom identique.

Complétion avec le gedcom courant

Quand cette option est activée, l'aide à la saisie utilise les noms, prénoms et professions du fichier Gedcom courant pour compléter les mots au fur et à mesure de la saisie.
Cette option est utile en particulier quand on commence un relevé et qu'il y a encore peu de termes disponibles pour la complétion de la saisie parmi les relevés eux-mêmes.

Remarque : Il est préférable de choisir un gedcom contenant des données d'un lieu et d'une date proche des relevés à saisir pour que la complétion puisse porposer des données semblables à celles qui doivent être saisies.
