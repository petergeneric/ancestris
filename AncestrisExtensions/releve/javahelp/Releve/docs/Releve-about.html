Relevé


 

Auteurs: Michel Pujol

Description de l'outil

Cet outil est destiné à saisir les actes en masse lors du dépouillement des registres  communaux ou notariaux.

Il est conçu pour faciliter la saisie des données et contrôler au fur et à mesure leur cohérence.

Les données sont enregistrées par commune.

un fichier  = une commune 


L'outil est composé de :

*  Quatre tables qui donnent une vue d'ensembles des relevés (voir Utilisation des tables)
  • la table des naissances
  • la table des mariages
  • la table des décès
  • la table des actes divers tels que les publications de mariage, contrats de mariage, testaments , quittances, etc.

* Un panneau de configuration pour préciser la commune du relevé et les options (voir Configuration )

* Un éditeur qui permet de créer ou modifier les relevés. (voir Utilisation de l'éditeur)

* Un second éditeur dans une fenêtre volante de taille suffisamment réduite pour être juxtaposé à une photo sur un écran lorsqu'on fait le dépouillement d'un registre numérisé (voir Utilisation de l'éditeur).

* Un menu d'enregistrement des données dans des fichiers qui sont consultables avec d'autres outils tel que des tableurs ou éditeur de texte, ou d'autres outils de gestions de relevés (voir gestion des fichiers)

  

* une fonction de copie des relevés vers un fichier gedcom  (voir Copie des relevés dans un fichier Gedcom)

* un outil de recherche des noms ou prénoms présents dans les relevés.


A quoi peut servir un relevé ?

Partage des informations
Le fichier des relevés peut être communiqué à une association de généalogie qui l’insère dans une base de données partagée par exemple par internet . On peut ainsi faire des recherches croisée avec les millions de relevés faits d'autres bénévoles.
Le menu Exporter permet de constituer le fichier dans le format requis par certaines associations.

Utilisation personnelle avec un autre outil
Le fichier de relevé peut s’ouvrir avec un tableur (openoffice, excel,..). On peut ainsi trier par colonne, ou fusionner, masquer, etc. les données affichées dans les colonnes distinctes avec toutes les fonctionnalités du tableur.

Copie des relevés dans un fichier GEDCOM
Voir la rubrique Copie des relevés dans un fichier GEDCOM dans l'aide du plugin. 
Comme le relevé ne contient que les informations principales, il est nécessaire de compléter les notes du gedcom en saisissant les informations supplémentaires des actes après avoir copié..

Premiers pas avec un fichier de démonstration

Pour faire les premiers pas, il est possible d'utiliser un exemple de données fourni avec l'outil. 

Cliquer sur le bouton Relevé Démo dans le menu fichier.  Un nouveau relevé est créé avec des exemples de données qu'il est possible de modifier, enregistrer, etc.
