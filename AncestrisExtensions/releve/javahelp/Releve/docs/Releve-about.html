Relevé


 

Auteurs: Michel Pujol

Cet outil est destiné à saisir les actes en masse lors du dépouillement des registres  communaux ou notariaux.

Il est conçu pour faciliter la saisie des données et contrôler au fur et à mesure leur cohérence.


Description de l'outil


Les données sont enregistrées par commune .

un fichier  = une commune 


L'outil est composé de :

*  Quatres tables qui donnent une vue d'ensembles des relevés (voir Utilisation des tables)
  • la table des naissances
  • la table des mariages
  • la table des décès
  • la table des actes divers tels que les publications de mariage, contrats de mariage, testaments , quittances, etc

* Un panneau de configuration pour préciser la commune du relevé et les options (voir Configuration )

* Un editeur qui permet de créer ou modifier les relevés. (voir Utilisation de l'éditeur)

* Un second editeur dans une fenêtre volante de taille suffisemment réduite pour être juxtaposé à une photo sur un écran lorsqu'on fait une dépouillement d'un registre numérisé (voir Utilisation de l'éditeur).

* Un menu d'enregistrement des données dans des fichiers qui sont consultables avec d'autres outils tel que des tableurs ou editeur de texte , ou d'autes outils de gestions de relevés (voir gestion des fichiers)

  

* un outil de recherche des noms ou prénoms présents dans les relevés.



Premiers pas 

Pour faire les premiers pas , il est possible d'utiliser un exemple de données fourni avec l'outil. 

Cliquer sur le bouton Relevé Démo dans le panneau de configuration .  Un nouveau relevé est créé avec des exemples de données qu'il est possible de modifier , enregistrer, etc ...
