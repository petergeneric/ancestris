Format des Fichiers NIMEGUEV3

Les quatre types de relevés ont des formats différents.

  • Les lignes des naissances contiennent 30 colonnes
  • Les lignes des mariages contiennent 60 colonnes
  • Les lignes des décès contiennent 38 colonnes.
  • Les lignes des actes divers  contiennent 64 colonnes

Les champs sont séparés par un point-virgule. Le dernier champ de la ligne est toujours suivi d'un point-virgule.
Les lignes sont séparées par  2 caractères x0A x0D
Si le caractère séparateur point-virgule est  présent à l'intérieur d'un champ, il est remplacé par un point lors de l'écriture dans le fichier car NIMEGUE ne supporte pas la recommandation  RFC 4180.
