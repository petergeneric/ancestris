Format des fichiers de relevé ANCESTRIS

Les quatre types de releves ont  tous le même format.
Les lignes sont contistutées de 78 champs séparés par un point-vigule.
Chaque ligne est terminée par le separateur "\r\n" .
Si le caractere séparateur point-virgule est présent à l'intérieur d'un champ , le champ est mis entre guillements lors de l'écriture dans le fichier. De même, si le caractère guillemet est présent à l'intérieur d'un champ, un deuxième guillement est ajouté après celui-ci (cf RFC 4180).
Certains champs sont ignorés à la lecture du fichier par Ancestris en fonction du type d'evenement, voir le tableau ci-dessous.


Astuce :  Il est possible de créer un fichier au format ANCESTRIS avec un tableur pour, par exemple, réutiliser des données déjà saisie avec un tableur
  1. créer un tableau de 78 colonnes avec le tableur,
  2. mettre ANCESTRISV1 dans la première colonne de toutes les lignes.
  3. renseigner  les autres colonnes en respectant la défintion ci-dessous. Il est possible de laisser des cellules vides excepté la1ère et la 8ième colonne.
  4. enregistrer les données au format CSV ou TXT ( avec point virgule comme séparateur de cellule)
