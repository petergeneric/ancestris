Edition d'un lieu


Fenêtre d'édition d'un lieu


Cette fenêtre permet de modifier plusieurs informations se rapportant aux lieux :
  • Bouton Gedcom : L'ordre des juridictions des lieux dans votre généalogie : tous les lieux du fichier Gedcom sont sur ce format-là. Ceci donne un nom à chacune des jurisdictions. En terme de fichier Gedcom, c'est l'enregistrement "PLAC" dans l'entête du fichier que toutes les étiquettes "PLAC" devront suivre.

  • Préférences : les préférences en général sur les composants d'un lieu : les juridictions, leur séquence, et si elles sont obligatoires ou non. Le bouton fait apparaître la fenêtre de préférence d'Ancestris comme si vous y accédiez depuis le menu principal. Ceci permet à Ancestris de savoir la signification de chaque jurisdictions d'un lieu.
    • Attention : si la liste des juridictions de votre généalogie n'est pas la même que celle des préférences, l'éditeur de lieu va peut-être placer les juridictions aux mauvais endroits.

      Par exemple, si l'on ouvre un fichier où se trouve

      lieudit,commune,code_insee,departement

      dans le format des lieux du Gedcom (bouton Format Gedcom), et que les préférences de lieux disent

      Lieudit:1, Commune:2, Code Insee:3, Code Postal:4, Département:5

      alors la fenêtre d'édition des lieux fera apparaître la juridiction "Département" dans la case "Code Postal" de la fenêtre, car c'est la 4è juridiction dans cet exemple.
      Le mieux dans ce cas est de temporairement changer les préférences de lieux d'Ancestris pour s'aligner sur le Gedcom.
      Si vous souhaitez faire l'inverse, c'est-à-dire réaligner les juridictions du Gedcom à celles de vos préférences Ancestris, alors il vous faudra passer les lieux à la moulinette de la boîte à outils d'Ancestris.

  • Description à conserver : les jurisdictions d'un lieu dans le fichier Gedcom que vous pouvez les changer avant d'appuyer sur le bouton OK.

  • Latitude et longitude : les coordonnées géographiques d'un lieu dans le fichier paramètre en local sur votre ordinateur. A vérifier avant d'enregistrer.


Au lancement de la fenêtre du lieu, c'est la partie gauche de la fenêtre qui est utilisée. Le lieu est recherché sur Internet et les réponses possibles sont affichées dans la liste. Sélectionnez le lieu qui convient et cliquez sur le bouton "Choisir ce lieu" pour le faire apparaître dans la partie droite de l'éditeur de lieu.
Si le lieu n'est pas trouvé, vous pouvez éventuellement changer son nom dans le champ en haut à gauche et relancer la recherche en appuyant sur le bouton "Rechercher".
