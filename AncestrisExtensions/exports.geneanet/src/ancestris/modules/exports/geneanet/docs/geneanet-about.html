Exports - Geneanet - Mode d'emploi

Geneanet :

Vous trouverez Geneanet à www.geneanet.org

Mode d'emploi:

Commencer par le réglage de Exports - Geneanet

C'est Outils /Préférences /Extensions /Export_Geneanet

Cocher les cases voulues.
Le fichier de log (ou journal) sert à enregistrer toutes les actions effectuées, c'est un fichier texte à consulter avec un éditeur de texte, il a pour adresse :
avec windows C:\Users \IdUtilisateur \AppData(caché) \Roaming \.ancestris \n°version \ancestris \ancestris.log
avec linux home /.ancestris(caché) /n°version /ancestris /ancestris.log
n°version est trunk pour une version d'essai ou 0.8 ou ...

Les boutons [OK] / [Annuler] servent à Valider / Abandonner les choix.
Les boutons [Export] / [Importer] servent à Sauvegarder / Récupérer le fichier de configuration d'Ancestris.

Quitter ce tableau de réglage des préférences par [OK].

Choisir le gedcom en ouvrant par exemple l'arbre dynamique.

Ouvrir Exports - Geneanet dans le menu Fichier/Exporter/Geneanet.

Premier écran :

Onglet Filtrer par Entités : cocher les cases des entités à exporter.

Sur cet écran deux réglages à faire :
1/ Le fichier de sortie
Enregistrer dans : Pour choisir le dossier d'enregistrement. Sur l'exemple c'est le Bureau.
Nom du fichier : Pour choisir le nom du fichier si celui proposé ne vous convient pas, tout en gardant le suffixe .gw .
2/ Les entités concernées par la sortie

Deuxième écran :

Onglet Filtrer par Propriétés : cocher les cases ou remplir les zones décrivant les exclusions.

Sur cet écran un réglage :
3/ Les propriétés que vous ne voulez pas exporter.

Troisième écran :

Onglet Filtrer par Vues : cocher les cases.

Sur cet écran deux réglages :
5/ La population concernée.
1ère ligne choix entre le gedcom entier ou seulement la partie visible dans l'arbre dynamique.
6/ Les données concernées.
2nde ligne choix entre toutes les données ou seulement les publiques, pour les privées voir Outils /Préférences /Affichage /Données_privées.

Lancement de Exports - Geneanet:

Utiliser le bouton [Enregistrer] lance la création et l'enregistrement du fichier .gw .

C'est un fichier texte, il se consulte avec un éditeur de texte.
Il est prêt à être envoyé vers www.geneanet.org.
