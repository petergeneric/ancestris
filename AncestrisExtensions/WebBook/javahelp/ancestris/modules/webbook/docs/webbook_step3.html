Ce panneau permet d'inclure ou non les sections multimédia dans le Livre Web. Pour chaque type de média ci-dessous, il existe quelques options de personnalisation.

Sources

Si vous décidez de générer la section des sources, le menu de la page d'accueil incluera un chapitre "Documents" contenant un lien vers une section des sources de votre généalogie. Tout d'abord la liste de ces sources, puis les détails des sources, c'est-à-dire leur titre, leur lieu de stockahe, les textes et médias qui leur sont attachés, et les entités associées à ces sources.
Une option du panneau vous permet de choisir la façon dont vous voulez présenter les entités associées à chaque source : simplement en affichant leurs noms en lien vers ces entités ; avec en plus le texte de source que l'on pourrait avoir dans les informations contenues dans les entités elles-mêmes ; ou finalement avec à la fois le texte et les médias qui sont contenues dans les entités elles-mêmes.
Par exemple, si de nombreux individus sont associés à des sources dans votre généalogie, et si vous avez pour habitude de mettre les textes et médias des contenus des sources dans les fiches des individus eux-mêmes, vous préférez peut-être alléger les pages générées en n'y incluant pas ces textes et médias.

Media

Si vous décidez de générer la section des medias, le menu de la page d'accueil incluera un chapitre "Documents" contenant un lien vers une section des médias contenus dans votre généalogie. Tout d'abord la liste de ces médias, puis les médias eux-mêmes. Ces médias sont généralement des photos, des enregistrements sonores et des vidéos.

Cartes

Si vous décidez de présenter une carte géographique de votre généalogie, le Livre Web comportera dans la page d'accueil un lien vers une carte Google avec l'emplacement de tous les lieux de votre généalogie.
Pour que les lieux soient bien placés, il y a une condition nécéssaire : il faut avoir lancé le module Géo d'Ancestris au moins une fois. C'est en effet ce module qui déterminera les coordonnées géographiques de tous vos lieux. Reportez vous au module Géo si nécessaire.
Par ailleurs, pour pouvoir utiliser une carte Google sur Internet, il faut au préalable avoir récupérer une clé d'identification de Google. Vous devez indiquer cette clé dans l'assistant.
