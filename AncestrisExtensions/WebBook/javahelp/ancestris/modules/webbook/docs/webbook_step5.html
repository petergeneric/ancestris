Assistant - Panneau "Copie locale"


[Panneau précédent][Panneau suivant]

Ce panneau permet de spécifier le nom du répertoire local ou seront stockés les pages de votre site Web, avant de les remonter sur votre serveur Internet.

Il permet également de donner le nom d'un fichier log. Ce fichier conserve le détail de la génération du Livre Web.
