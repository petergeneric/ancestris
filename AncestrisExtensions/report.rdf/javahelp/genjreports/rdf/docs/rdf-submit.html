You can submit your rules and/or SPARQL queries with SVN to docs/reports but don't forget to update docs/rdf-examples.html.

There is more about SVN on the wiki-pages of the project. As an alternative you may choose to mail to one of the discussion lists, for someone else to pick up.

localization

To localize a query submit a variant of the general

You will need a stylesheet per query per language. Provide links to the special stylesheets on the example page.

The general style sheet contains something like: <xsl:for-each ...><th>...</th></xsl:for-each> replace the complete code snippet with localized column headings, for example: <th>Name</th><th>ID</th><th>Birth date</th> You might also want to change the <title> and <h1>.
