Exports - SiteWeb : Mode d'emploi

Mode d'emploi:

Commencer par le réglage de Exports - SiteWeb

C'est Outils /Préférences /Extensions /Site Web

Ne pas hésiter à élargir la fenêtre pour voir tout son contenu.
Ceci est un photo-montage, vous ne verrez jamais toutes ces lignes en même temps, mais un ascenseur à droite de la fenêtre permet d'accéder aux lignes cachées.
Certaines lignes sont des cases à cocher (ex ligne 1), cocher les cases voulues.
D'autres sont des menus déroulants(ex ligne 5).
D'autres attendent des textes(ex ligne 11).

Voici les réglages que j'ai utilisés pour faire le SiteWeb Bourbon
Formatage des lieux :all
Il semble que ce soit la seule valeur prévue pour le moment.
Texte de présentation de la page d'accueil du rapport :Exemple de site Web réalisé avec la Maison Capétienne de Bourbon
Titre de la page d'accueil :Ancestris Bourbon
Traduire vers la langue suivante (ISO639 2 lettres, ex en) :en
Choix pour cette 2ème langue(à ce jour) : da, de, el, en, es, fi, it, nl, pt, sv ou rien. Si vous ne voulez pas de 2ème langue, supprimer en, le site sera plus petit.
Type d'arbre : Parents à gauche, enfants à droite

Les boutons [OK] / [Annuler] servent à Valider / Abandonner les choix.
Les boutons [Export] / [Importer] servent à Sauvegarder / Récupérer le fichier de configuration d'Ancestris.

Quitter ce tableau de réglage des préférences par [OK].

Choisir le gedcom en ouvrant par exemple l'arbre dynamique.

Ouvrir Exports - SiteWeb dans le menu Outils/Multimédia/Export Site Web.

Il faut que le dossier existe afin d'y déposer tous les documents.

Sur cet écran, que vous reconnaissez,
c'est celui que vous avez lorsque vous faites enregistrer sous avec n'importe quel logiciel,
vous pouvez choisir ou créer l'emplacement du site.

Le site est un dossier (ou répertoire) de préférence vide Ex : SiteWeb que vous placez où vous voulez.

Le sélectionner et cliquer sur [OK] lance la création du site et ouvre dans Ancestris une fenêtre Documents où s'affichent toutes les opérations effectuées.

Utiliser le Site Web.

Ouvrir le dossier SiteWeb, celui que vous avez choisi ou créé tout à l'heure.

La porte d'entrée est index.html, un double-clic dessus l'ouvre dans votre navigateur habituel.
Si vous ne voyez pas .html c'est peut-être que vous l'avez caché, ce n'est pas grave.
S'il y a index-en.html c'est la version anglaise et index.html la version française, vous avez le choix.
 Bonne promenade dans votre site.

Pour le communiquer à quelqu'un, transporter tout le dossier et uniquement lui.

Le site créé à partir de la généalogie Bourbon fait 3.68Mo en 1 langue et 3.80Mo en 2 langues car les feuilles html sont créées une fois dans chaque langue et les photos ne figurent qu'une fois.

Pour envoyer le siteWeb par mail, il faut le zipper, l'envoyer puis à réception l'extraire où vous voulez.
