Travailler avec les divers Fenêtres de Visualisation

Une fois que vous avez ouvert un fichier contenant des données généalogiques au format Gedcom (*.ged), à partir du Centre de Contrôle vous pouvez ouvrir plusieurs types de visualisation.

Ces visualisations travaillent sur les mêmes données - cela veut dire que :

  • les changements effectués dans une fenêtre seront immédiatement effectifs dans les autres fenêtres pour le même fichier,
  • les évènements sélectionnés dans une fenêtre vont être transmis aux autres fenêtres pour le même fichier (*),
  • la fermeture d'un fichier entrainera immédiatement la fermeture de toutes les fenêtres ouvertes sur ce fichier,
  • les menus contextuels pour les informations affichées offrent des actions dans tous les types de vues.

      * les changements dans la sélection d'une fenêtre peuvent ne pas provoquer un changement dans une autre si cette dernière ne peut pas suivre la sélection

En général, vous ouvrirez une fenêtre par type de vue - pas nécessairement toutes à la fois, mais leur association vous aidera à naviguer et comprendre le fondement des données. Par exemple :

  • Fenêtres de Données & Arbre pour se promener à travers les informations existantes.
  • Fenêtres d'Edition & Données pour corriger, cataloguer, éditer les informations, et en ajouter.
  • Fenêtre de Scripts (Rapports), pour lancer des scripts sur des fichiers et en ressortir des informations importantes (statistiques, etc...).

Leur combination n'est pas limitée et trés rapidement vous trouverez par vous-même quelles vues vous est nécessaire pour satisfaire vos besoins. La position et la taille des fenêtres sont enregistrées, aussi placer les selon vos goûts et elles se reouvriront aux mêmes endroits la fois prochaine. Vous pouvez également ouvrir plusieurs instances d'une même vue pour vérifier des données à différents endroits - elles sont numérotées consécutivement.
