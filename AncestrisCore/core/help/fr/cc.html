Le Centre de Contrôle

La première chose que vous voyez aprés avoir démarré GenJ, est ce qu'on appelle le "Centre de Contrôle". Celui-ci présente une liste des fichiers Gedcom qui sont ouverts.
Sur l'exemple ci-dessous, un seul fichier est ouvert, il s'appelle "francois.ged", il comporte 40.897 lignes, et le programme occupe 16,5 megas en mémoire.


Vous disposez d'une liste d'icônes qui pourraient avoir les labels suivants :
Nouveau  -  Ouvrir  -  Table  -  Arbre  -  Chrono  -  Editer  -  Rappt   -  Navig   -  Entité   -  Recher  -  Géo.

Ces fonctions sont accessibles par un clic sur l'icône concernée mais aussi en entrant dans l'un des sous-menus "Fichier" et "Visualisations".
En effet, si vous regardez au dessus de ces icônes, vous trouverez trois menus déroulants accessibles avec votre souris, mais aussi en utilisant les raccourcis clavier (ex : Alt-f ouvre le menu déroulant "Fichier", Alt-v ouvre le menu "Visualisations").
Toujours dans le menu déroulant "Visualisations", vous allez pouvoir aussi accéder à un sous menu appelé "Options" permettant de paramétrer GenJ, notamment par exemple de choisir la langue, le nombre d'annulations, etc...

Lors de la première utilisation un fichier dénommé 'example.ged' devrait avoir été chargé automatiquement. Vous pouvez choisir de découvrir les diverses possibilités de GenJ en ouvrant l'une des vues sur ce fichier, ou créer votre propre fichier Gedcom (votre dossier généalogique) pour démarrer. Les boutons en haut de la fenêtre vous permettre d'accéder à toutes ces fonctions.

 Crée un fichier Gedcom,
 Ouvre un fichier Gedcom,
 Ouvre un Tableau de Données *,
 Ouvre la visualisation de l'Arbre Généalogique *,
 Ouvre une fenêtre montrant la Chronologie des Événements *,
 Ouvre la fenêtre d'Edition *,
 Ouvre la fenêtre de visualisation des Scripts (Rapports) *,
 Ouvre la fenêtre du Navigateur *,
 Ouvre la fenêtre de visualisation générale d'une Entité *,
 Ouvre la fenêtre de Recherche *,
 Ouvre la fenêtre de Localisation Géographique *,
    * (Assurez-vous que vous avez sélectionné le fichier sur lequel vous voulez travailler dans la liste des fichiers Gedcom disponibles)

En cliquant sur l'un des liens ci-dessus vous pourrez en apprendre plus sur la fonction concernée.

Vous pouvez ouvrir autant de vues que vous voulez - elles seront numérotées d'une manière séquentielle et chacune disposera de ses propres réglages, paramètres, préférences, au cas où vous souhaiteriez visualiser vos données de différentes manières.
Le Menu déroulant à partir de l'onglet "Fichier", permet d'ouvrir/enregistrer/fermer un fichier mais aussi de quitter GenJ.
Sinon, comme vous le voyez sur la capture d'écran ci-dessus, en laissant votre souris sur une icone, un petit texte apparaît pour vous donner une explication supplémentaire.
