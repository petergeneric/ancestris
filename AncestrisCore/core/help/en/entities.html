Entities

GenJ lets you model these entities (following the Gedcom standard):

 Individual
 Family
 Media
 Note
 Source
 Submitter
 Repository
To create entities right-click in any view to open a Context Menu. From there choose to add the desired entity for
  1. a property or
  2. an entity or
  3. a Gedcom file
Whether sub-menus for 1-3 are available depends on the context you clicked on. If you choose to create and add an entity for any of the above a dialog will let you confirm that a new instance is to be created (referring to existing instances is possible, too).
