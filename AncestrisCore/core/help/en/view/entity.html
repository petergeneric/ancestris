Views - Entity View

This view renders one entity and its properties into a single pane. This can be setup as the most complex rendering option that might not be feasible for the Tree View because of the resulting performance hit.

When selecting an entity in any of the other views this view will change its selection and render it as you browse the available entities.

Toolbar

The following actions are available:

  1. open settings for view
  2. close view

Settings

The settings for this view are grouped into 2 sections

  1. Main - rendering options
  2. Type Blueprints - choose Blueprints used for rendering entities
