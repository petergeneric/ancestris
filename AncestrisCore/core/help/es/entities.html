Entidades

GenJ proporciona estas entidades (siguiendo el estándar Gedcom):

 Individuo
 Familia
 Medio
 Nota
 Fuente
 Remisor
 Repositorio
Para crear entidades haga un clic derecho en cualquier vista para abrir el Menu contextual. Ahora elija la entidad para
  1. una propiedad o
  2. una entidad o
  3. un fichero Gedcom
los submenús están disponibles en función del contexto en el que haga clic. Si elije crear y añadir una entidad para cualquiera de esas opciones se le pedirá confirmación para crearla o hacer referencia a entidades ya existentes.
