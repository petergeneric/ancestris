Usando LifeLines

LifeLines no está directamente soportado por GenJ. Necesitará un programa adicional llamado GDBI para ello. GDBI es la abreviatura de Gedcom DataBase Interface y puede ser integrado con varios programas genealogicos para proporcionar características adicionales. GenJ es uno de esos programas y puede asi realizar informes con LifeLines.

Instalación, y arranque del programa

La instalación para el uso con GenJ se explica en la documentación de GDBI (tambien accesible a través de la página inicial de GDBI). El fichero necesario es genj.html. Dos notas:

  1. Puede tambien usar un programa que soporte el formato jar para descomprimir las clases inform (ReportBK.class, ReportLL.class), p.e. WinRAR.
  2. Para comenzar GenJ con GDBI debe ejecutar el fichero por lotes (*.bat) creado durante la instalación y no el script de inicio de GenJ como es usual.

Trabajando con GDBI

Después de instalar e iniciar el programa dos nuevas entradas aparecen en la vista de informes. Las entradas empiezan por GDBI:

Sólo GDBI: LifeLines report es relevante para hacer informes LifeLines . Despues de iniciar este informe un dialogo de selección de informes aparece. Despues el GDBI procesa el informe y muestra el resultado.
