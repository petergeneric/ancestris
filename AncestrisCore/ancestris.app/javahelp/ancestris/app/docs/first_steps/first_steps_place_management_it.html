E' un capitolo importante. Una attenta lettura vi eviterà in futuro, di dover modificare tutto il vostro lavoro. L'adozione di un ordine universale ed omogeneo delle giurisdizioni, conforme alla norma Gedcom, è indispensabile per facilitare i vostri scambi e render più efficace la geolocalizzazione degli eventi.



ATTENZIONE : prima di ogni modifica salvate i vostri files Gedcom.

L'ordinamento delle giurisdizioni

Per censire un luogo conviene classificare le differento giurisdizioni per ordine crescente d'importanza, separando ogni livello con una virgola. Anche se un livello non viene descritto mantenere la virgola al posto del livello mancante.

Consiglio di Ancestris : contrariamente a cerune pratiche, non lasciare spazi tra le giurisdizioni.

Sette livelli di giurisdizione sono possibili :
Frazione,Comune,Codice_postale,Provincia,Regione,Paese

Non siete obbligati a definirli tutti, ma bisognerà rispettare la gerarchia del loro ordinamento.

Bien qu'Ancestris permette de saisir aussi le code postal, il est préférable d'utiliser le code INSEE. En effet, un code postal peut regrouper plusieurs communes alors que le code INSEE (système de code Géographique créé en 1943), est unique pour chaque commune.


Il censimento delle giurisdizioni

Si vous avez opté pour la présentation "distinguer les juridictions" (cf Les premiers paramètres) vous n'avez pas à vous soucier des explications qui suivent, vous saisirez les juridictions niveau par niveau.
Pour ceux qui n'ont pas choisi cette option, le plus simple, c'est un exemple:
Pour un événement localisé au lieu-dit "Mézin", à Allanche (code INSEE 15160), département du Cantal, en Auvergne, France. Il convient de saisir:

  • On ne met rien après le mot France.
  • Si le lieu-dit est inconnu on met une virgule en début de ligne.
  • Pour un événement dont on ne connaît que la commune, on écrit: ,Allanche,,,, (1 virgule en tête et 4 virgules en fin de ligne).

C'est simple, retenir: "Du plus petit au plus grand et chaque terme séparé par une virgule".

Auto-complètement de champ

Pendant la saisie, un menu déroulant s'ouvrira vous proposant les lieux déjà connus commençant par la même chaîne de caractères. → Un clic gauche sur la ligne proposée et OK pour valider.
Si vous avez choisi pour la saisie "Préférences - juridictions séparées", cette fonction existe également mais ligne par ligne.


Ne pas confondre l’enregistrement dans le fichier Gedcom et l'affichage des lieux dans les vues

L’enregistrement dans le fichier Gedcom, c'est là que la norme d'organisation des juridictions devra être respectée.
L'affichage des lieux dans les vues, c'est la façon dont vous voulez voir s'afficher les juridictions dans les vues (par exemple, choisir de n'afficher dans l'arbre que la ville et le pays - cf Préférences).

La registrazione dell'ordine delle giurisdizioni nel file Gedcom

Il est important que cette hiérarchie des lieux (ou ordre des juridictions) soit décrite dans le header (l'entête) de votre Gedcom. C'est la règle interne fixée pour le fichier Gedcom qui permet à Ancestris et aux autres utilisateurs de lire les juridictions. Cet enregistrement détermine le format de toutes vos étiquettes "PLAC".
Cet enregistrement devra être fait dès la création d'une généalogie ou suite à l'importation d'une généalogie créée avec un autre logiciel (beaucoup ne respectent pas la norme).

  • 1ère solution:
    • Ouvrir l'éditeur Gedcom. Pointer une ligne PLAC (au choix)
      - clic gauche (pour la mettre en surbrillance) puis clic droit pour ouvrir le menu contextuel .
      - descendre sur la ligne Propriété Lieu-... (correspondant à votre choix)... un nouveau sous-menu apparaît. Clic sur la première ligne ''Établir la hiérarchie des Entités juridictionnelles''

      La ligne du bas sert à contrôler, modifier ou saisir les différentes juridictions que vous avez retenues pour toute votre généalogie.
      Pour valider votre choix: cliquer sur le bouton "Exécuter et Établir la hiérarchie des entités juridictionnelles".
      C'est fini! Pensez à sauvegarder votre travail.

  • 2ème solution:
    • Ouvrir votre fichier Gedcom avec un éditeur de texte. Pour Linux, utilisez Gedit, ou Nedit, ou Jedit. Sous Windows, utilisez Notepad (surtout pas Word). Sous MacOs, utilisez  TextEdit.
      •         0 HEAD
        1 SOUR ANCESTRIS
        2 VERS 0.5.0.1291
        2 NAME Ancestris
        2 CORP Ancestris Team
        3 ADDR http://www.ancestris.org, http://www.ancestris.com
        1 DEST ANY
        1 DATE 21 MAR 2011      <---- La date sera différente
        2 TIME 18:44:12         <---- L'heure sera différente
        1 SUBM @B1@             <---- Peut être que vous n'aurez pas cette ligne
        1 FILE francois.ged     <---- Vous aurez là le nom de votre fichier
        1 GEDC
        2 VERS 5.5
        2 FORM Lineage-Linked
        1 CHAR UTF-8
        1 LANG French

      Il vous suffit de rajouter, juste après la ligne ''1 LANG French'', les deux lignes suivantes: 

              1 PLAC
        2 FORM Lieudit,Commune,Code,Département,Région,Pays
      Sauvegardez votre travail, et voilà, c'est fini!

      Si vous avez déjà saisi des centaines de lieux dans votre fichier, le simple fait de rajouter dans l'entête du fichier Gedcom les deux lignes décrites dans le paragraphe précédent ne changera rien à tous les lieux déjà enregistrés. La seule solution sera de reprendre un par un tous les lieux de votre généalogie.


  • une autre solution pour gérer l'ordonnancement des juridictions (si l'on hésite à modifier avec un éditeur de texte) est d'aller dans "Affichage /Listes et rapports/ Boîte à outils/ 5 - Gestion des étiquettes d'adresses PLAC".

    • Le format des lieux de votre fichier Gedcom ainsi que vos préférences d'affichage peuvent également être modifiés à partir du plug-in "Liste des lieux".


Modifica globale del nome delle giurisdizioni

Vous vous apercevez que depuis le début, vous faites une erreur de frappe sur une localité. Autre hypothèse, une même localité a été enregistrée de façons différentes.
Il suffit de quelques clics pour modifier en une fois tous les lieux concernés:

    • surligner une des lignes PLAC que vous souhaitez modifier (1)
    • faites la modification dans la zone de saisie (2)- dans cet exemple on supprime le "s" - ce qui fera apparaître un petit carré au bout de la ligne (3)
    • cochez ce carré puis "oui" dans la fenêtre de demande de confirmation (4)
    • il ne vous reste plus qu'à cliquer sur ok dans la zone basse de l'éditeur pour valider ce changement.


      • Cette modification est identique à partir de l'un ou l'autre des deux éditeurs.
