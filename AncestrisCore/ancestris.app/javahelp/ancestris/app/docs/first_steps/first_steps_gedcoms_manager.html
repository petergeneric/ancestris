Gedcom Explorer


Gedcom Explorer allows you to navigate between your different Gedcom files ; it sums up your entities and shows it's content.
Particularities of this window : it can not be undocked nor closed. It's default position is hidden but you can make it a Tab. (cf. Manage Windows).




  • A menu proposes 2 types of information :
    • All entities;
    • Close relatives (this function will be active on a future version).

The tree
For each of your open Gedcom files, each line (or entity level) can be unfolded to show the details of its content :
  • The number of individuals
    • list of all individuals in alphabetical order;
  • The number of famillies
    • list of all famillies in alphabetical order;
  • The number of sources
    • details of created sources;
  • The number of repositories
    • details of all registered repositories
  • The number of Notes **
    • details of your notes;
  • The number of information suppliers
    • details of your information suppliers;
  • The number of multimedia elements
    • details of your multimedia elements.

** The Notes reported here are only the "entities notes" (generated from a right click on an entity, followed by "Gedcom xxxxx.ged, create a note"), with the exception of "notes for elements of an entity" or "notes for elements of a property of an entity" (note attached to BIRT, MAR or other). These entity notes can be useful whenever information is recurrent or respecting several individuals (ex : filiation to be proved).
