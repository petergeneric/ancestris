Utilisation de base :

Vous venez de télécharger et d'installer Ancestris.
Cette page va vous guider parmi les didacticiels d'utilisation d'Ancestris.
Avant de commencer votre généalogie, dans le paragraphe suivant, nous aborderons quelques conseils pour vous familiariser avec Ancestris,
puis nous verrons comment importer une généalogie et enfin comment ajouter de nouvelles données.

Démarrer avec Ancestris

  1. L’encodage des fichiers Gedcom. Ancestris respecte au plus près la Norme, quelques remarques à son sujet.

  2. La gestion des lieux. Des recommandations pour ne pas avoir à reprendre votre travail plus tard !

  3. Apprendre à manipuler les Fenêtres. Ancestris est très personnalisable.

  4. L'Explorateur : savoir comment naviguer entre vos fichiers de généalogie.

  5. De nombreuses fonctionnalités additionnelles, indispensables : Les Plugins !


  • Pensez à utiliser les exemples de Généalogie proposés via l'écran de bienvenue (capture d'écran ci-dessous).
    C'est un bon moyen pour apprendre à utiliser Ancestris sans abîmer votre généalogie.


  • Vous pourrez retrouver, en détails, l'aide sur chaque fenêtre
    les éditeurs, les navigateur, l'arbre dynamique, la table, etc... (capture ci-dessous).


  • Pour aller plus loin, voir l'aide de la fenêtre de configuration via : menu Outils / Option + F1
