How to start using Ancestris

Ancestris is a configurable software. Therefore, for a better use, we advise you to start by visiting the chapter "The First Parameters".

And now, a few tips to help you get started with the software:
  • Gedcom file encoding. A few notions on how Ancestris respects the standard as closely as possible.
  • Window management. How to customize your environment.
    Find detailed help on each window.
  • The Explorer. How to browse through your files.
  • Place management. Some recommandations in order to avoid starting your work over in future!
  • Plugins. Additional features.

A good way to learn how to use Ancestris without damaging your genealogy file? Use genealogy examples offered on the welcome screen.


(The welcome screen allows you to start using Ancestris: open genealogy examples and have access to the main Ancestris Wiki help pages.)

How to build your genealogy

When Ancestris launches, the "My Ancestris" starting page / tab offers the possibility to use a wizard that will guide you step by step in the creation of your genealogy file.

For those who prefer an alternative method:

  • Adding a spouse

  • Adding children
