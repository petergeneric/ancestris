Migrazione da un altro programma di genealogia


Se avete già una genealogia creata da un programma e avete scelto di usare Ancestris.

La migrazione si fa in due tappe:

  • 1/ Preparazione: partendo dal vostro programma precedente, bisogna esportare i vostri dati in un file di formato GedCom (suffisso .ged).
    Ogni buon programma permette questa operazione. Se incontrate dei problemi (spesso legati al non rispetto della norma Gedcom) consultare il capitolo seguente: Problemi incontrati durante l'importazione.
  • 2/ Importare i vostri dati in Ancestris.
    • Partendo da barra del Menu "File / Apri", individuate il vostro file xxxx.ged.
    • selezionarlo, poi OK.


Attenzione: Prima di qualunque altra operazione verificate adesso, l'ordinamento delle giurisdizioni.
