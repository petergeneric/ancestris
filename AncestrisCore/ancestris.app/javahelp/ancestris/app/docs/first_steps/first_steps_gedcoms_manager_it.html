L'explorateur Gedcom


L'explorateur gedcom vous permet de naviguer entre vos différents fichiers Gedcom ; il résume les entités et vous informe de leur contenu.
Particularité de cette fenêtre : elle ne peut pas être désancrée. Elle ne peut pas être fermée. Sa position naturelle est d'être escamotée mais vous pouvez l'installer en onglet. (cf. La Gestion des fenêtres).




  • Un menu vous propose 2 types d'information :
    • Toutes les entités;
    • Parenté proche (cette fonction deviendra active dans une prochaine version).

L'arborescence
Pour chacun de vos fichiers Gedcom ouverts, chaque ligne (ou niveau d'entité) peut être déverrouillée pour donner le détail de son contenu :
  • Le nombre d'Individus
    • la liste par ordre alphabétique de tous les individus,
  • Le nombre de Familles
    • la liste par ordre alphabétique de toutes les familles,
  • Le nombre de Sources
    • le détail de toutes les sources créées,
  • Le nombre de lieux de stockage
    • le détail de tous les lieux de dépôt enregistrés,
  • Le nombre de Notes **
    • le détail de vos notes,
  • Le nombre de fournisseurs d'information
    • le détail de vos fournisseurs d'information,
  • Le nombre d'éléments multimédias
    • le détail des éléments multimédias utilisés.

** Les Notes répertoriées ici sont uniquement les "notes entités" (générées à partir d'un clic droit sur une entité puis "Créer ou établir une liaison vers une note"), à l'exception des "notes sous-éléments d'une entité" (note rattachée à BIRT, MAR ou autre). Ces notes entités peuvent être utiles lorsqu'une information est récurrente ou lorsqu'elles concernent plusieurs individus (ex : filiation à vérifier).
