Informazioni generali sui plugins


I "Plugins" (moduli o programmi aggiuntivi o innesti) sono dei programmi non autonomi, che si aggiungono al programma principale di Ancestris per fornire nuove funzioni.

I plugins già installati e quelli disponibili
Si trovano in "Strumenti" / "Plugins". I numeri tra parentesi del 2° e 4° pulsante indicano il numero di plugins.
Per ogni Plugin selezionato, nella finestra di destra si hanno le informazioni relative.

Download e installazione
Il suffit de quelques clics pour installer un plugin (ou plusieurs), à partir de l'onglet "Plugins disponibles (x)".



  • Recharger le catalogue pour obtenir une mise à jour de la liste proposée.
    Pour éviter cette étape : dans l'onglet "Paramètres", allez dans la partie basse sur "Rechercher automatiquement les mises à jour -Intervalle de recherche-" et optez pour "A chaque démarrage".
  • Installation:
    • Cochez la case correspondant à votre choix. Le bouton "Installer" va devenir actif.
    • Clic gauche, la fenêtre "Installateur de plugins" s'ouvre, elle résume les plugins éventuellement associés.
    • Confirmez votre intention en cliquant sur "Suivant". Une fenêtre vous donne les conditions légales d'utilisation de ce plugin. Pour poursuivre l'installation, vous devez accepter les termes de toutes les licences associées au(x) plugin(s) sélectionné(s).
    • Cochez la case, en bas à gauche, "J'accepte tous les termes du contrat de licence". Le bouton "Installer", devient actif.
    • Clic gauche sur "Installer". Le plugin est téléchargé. Une fenêtre "Avertissement de validation" vous informe de la signature (ou non) du plugin. Quelque soit la présence ou non de cette signature, aucun risque, ce plugin vient bien du site officiel d'Ancestris.
    • Cliquez sur le bouton "Continuer".
L'installation est terminée. Dans certains cas il sera nécessaire de redémarrer le programme avant l'utilisation de ce nouveau plugin.

Désinstaller un plugin
Vous retournez dans "Outils / Plug-ins" pour ouvrir, cette fois, l'onglet "Installés (x)".

    • cochez la case devant le plugin que vous voulez désinstaller. Le bouton "Désinstaller" devient actif.
    • cliquez sur "Désinstaller". La fenêtre "Installateur de plugins" s'ouvre, elle résume les plugins associés qui vont être désinstallés.
    • confirmez votre choix.
Après cette suppression, vous aurez le choix de redémarrer Ancestris "immédiatement" (solution préférable) ou "plus tard". Votre choix étant fait, cliquez sur "Terminer".



N’hésitez pas à tester des plugins, leur désinstallation est facile, complète et rapide.
