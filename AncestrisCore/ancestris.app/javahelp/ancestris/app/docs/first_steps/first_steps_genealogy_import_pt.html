IMPORTAR UMA GENEALOGIA


Importar uma genealogia significa simplesmente carregar uma genealogia no Ancestris.

O Ancestris é um programa que lê um ficheiro em formato de texto em que a arquitectura é definida pela norma Gedcom (padrão internacional de troca de informações entre os programas de genealogia). Muitos programas não respeitam escrupulosamente esta norma (utilização de instalações, de parâmetros que lhe são específicos,...). Consequência: a perda de parte da informação durante a troca entre genealogistas. Foi por isto que o Ancestris fez a escolha de ser 100% Gedcom!

Para importar uma genealogia no Ancestris, duas situações:

 O seu outro programa é capaz de exportar os seus dados em formato Gedcom. O Ancestris lê-os sem problemas.
 O seu outro programa não é capaz de exportar os seus dados em formato Gedcom, excepto para criar um « motor  » para produzir um formato Gedcom, deverá reintroduzir os seus dados diretamente no Ancestris.

Mesmo na primeira situação, como a quase totalidade dos programas só respeitam parcialmente a norma Gedcom, o seu ficheiro original está provavelmente imperfeito. É aqui que começa verdadeiramente a apreciar o Ancestris:

 O Ancestris aceita ler não importa que ficheiro, mesmo que não esteja conforme.
 As ferramentas específicas do Ancestris foram criadas para lhe permitir corrigir certos erros dos outros programas. Não poderemos corrigir tudo, mas de certeza que conseguiremos corrigir alguns.
