Visualisation Chronologique


Cette vue affiche chronologiquement tous les évenements (des types que vous avez sélectionnés) de votre fichier, de gauche (les plus anciens) vers la doite (les plus récents).
Tous les évènements saisis pour les individus et familles, dans toutes les possibilités du format GEDCOM sont ainsi visualisables. Les évènements visualisés sont représentés avec une icône symbolisant le type d'évènement et le nom de la personne à laquelle cet évènement est associé.
Quand il y a un trop grand nombre d'évènements à afficher pour une même période, les évènements s'ajoutent de haut en bas pour être tous lisibles; des ascenseurs en bas et à droite permettent d'accèder à tous les évènements.

La sélection d'un évènement dans cette fenêtre, met à jour les données affichées dans les autres fenêtres:
  • Editeurs
  • Table
  • Arbre
  • Navigateur
  • Entité
Cette vue permet d'utiliser le menu contextuel
Ce menu est disponible par un clic droit sur un évènement.

Liste des différents outils disponibles dans le menu à droite:
Toutes les modifications apportées à cette liste peuvent être validées (bouton "appliquer") ou abandonnées (bouton "rétablir" ou directement "fermer").
  • Réglage du zoom
  • Réglage des paramêtres :
    • Onglet "généraux" :
      • Partie haute :
        Liste des différents évènements visualisables dans la vue : cases à cocher.
      • Partie basse :
        Options : afficher les abbréviations :
        Affiche/masque l'étiquette GEDCOM de l'évènement entre le symbole et le nom de la personne.
        Affiche/masque les dates à côté du symbole.
        Affiche/masque l'affichage d'un quadrillage annuel.
        Personnalisation des écarts avant et après les évènements.
    • Onglet "couleurs" :
      • Personnalisation des couleurs des textes, pointeurs, étiquettes, contenu, quadrillage et date.
