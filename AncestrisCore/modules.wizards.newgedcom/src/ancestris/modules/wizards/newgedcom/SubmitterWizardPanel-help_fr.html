Auteur de cette généalogie

Les informations demandées dans cet écran sont obligatoires pour tout fichier de généalogie. Il vous faut les remplir avec soin.

Nom complet

Il vous faut mettre ici votre prénom et votre nom.
  • Exemple: Marcel DUPONT
  • Pour les femmes mariées, vous êtes libres de mettre ce que vous voulez. Par exemple, si vous êtes née DURAND, mariée avec un Monsieur DUPONT, vous pouvez mettre : Jeannine DUPONT, ou Jeannine DUPONT née DURAND, ou Jeannine DURAND.

Adresse

L'adresse est sous forme postale.
  • Exemple: B.P. 327
    Résidence des Tamaris
    Allée des Cathares

Code Postal

Pas besoin d'explication.

Ville

Attention à l'orthographe exacte du nom. Ne pas oublier les petits tirets si le nom de la ville est un nom composé.
  • Exemple: Saint-Étienne-de-Mer-Morte

Etat

En France nous ne sommes pas concernés, mais si vous êtes américain, cela pourrait-être New Mexico, par exemple.
